**GEOLOG GASRATIO@Pixler** — настольная программа для загрузки, просмотра,
редактирования, интерпретации и подготовки к печати геолого-технологических,
газокаротажных и скважинных данных.

Программа предназначена для работы с материалами ГТИ, ГИС и газового каротажа,
создания многотрековых планшетов, расчёта производных кривых, описания литологии,
выделения интервалов и подготовки мастерлогов.

## Основные возможности

- импорт LAS, CSV, TXT и Excel;
- просмотр скважинных данных на многотрековом планшете;
- работа с глубинными, временными и пользовательскими индексами;
- редактирование кривых, заголовков LAS и метаданных;
- расчёт газовых отношений и пользовательских формул;
- работа с компонентами C1–C5 и нормализованными газовыми кривыми;
- создание литологических колонок, описаний пород и стратиграфии;
- создание интерпретаций и глубинных интервалов;
- отображение интервалов на планшете и редактирование их свойств;
- Undo/Redo для операций редактирования;
- сохранение проектов без изменения исходных файлов;
- экспорт данных и интервалов в LAS, JSON, CSV и Excel;
- подготовка планшетов, мастерлогов и печатных материалов;
- интерфейс на русском, казахском и английском языках.

## Запуск

Требуется Python 3.11 или новее.

### Windows

```powershell
py -3.11 -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -e ".[dev]"
geolog-gasratio-pixler
```

### Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e ".[dev]"
geolog-gasratio-pixler
```
