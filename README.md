# GEOLOG GASRATIO@Pixler

Отдельное расширяемое desktop-приложение для LAS/ГИС, ГТИ, газового каротажа,
литологии, шламограммы, стратиграфии, корреляции и печати мастерлогов.

## Инкремент 0.2

Работает:

- главное окно PySide6;
- загрузка LAS;
- дерево скважины, набора и кривых;
- интерактивный просмотр кривых через PyQtGraph;
- базовые арифметические Gas Ratio: C1/C2, C1/C3, C2/C3, C1/(C2+C3);
- расчётная сумма доступных C1–C5-компонентов;
- повторный пересчёт производных кривых;
- безопасное атомарное сохранение проекта в JSON;
- реестр версионируемых формул Pixler без непроверенных встроенных формул;
- автоматические тесты ядра.

## Запуск

```powershell
py -3.11 -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -e ".[dev]"
geolog-gasratio-pixler
```

## Важное ограничение

Формулы Pixler, Wetness, Balance, Character, флюидности и D-экспоненты будут
добавляться только с подтверждённым источником, единицами и контрольными примерами.
Это исключает скрытое использование неправильной методики.

См. `docs/` для полного плана.
