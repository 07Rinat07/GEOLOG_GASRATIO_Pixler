# GEOLOG GASRATIO@Pixler

Расширяемое desktop-приложение для работы с LAS/ГИС, данными ГТИ, газовым
каротажем, литологией, шламограммой, стратиграфией, корреляцией скважин и
печатными мастерлогами.

## Текущий инкремент 0.3

Реализовано:

- создание, открытие и атомарное сохранение проектов;
- загрузка нескольких LAS-наборов в проект;
- дерево скважин, наборов данных и кривых;
- выбор текущего набора и отдельной кривой из дерева;
- интерактивный просмотр кривых через PyQtGraph;
- базовые Gas Ratio: C1/C2, C1/C3, C2/C3, C1/(C2+C3);
- расчётная сумма доступных компонентов C1–C5;
- повторный пересчёт производных кривых;
- версионируемый реестр формул Pixler;
- расширенный `.gitignore` для зависимостей, окружений, кэшей и локальных данных;
- автоматические тесты ядра и сериализации проекта.

## Запуск в Windows

```powershell
py -3.11 -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -e ".[dev]"
geolog-gasratio-pixler
```
