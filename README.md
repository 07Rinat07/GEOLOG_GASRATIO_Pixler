**GEOLOG GASRATIO@Pixler** — настольная программа для загрузки, просмотра,
редактирования, интерпретации и подготовки к печати геолого-технологических,
газокаротажных и скважинных данных.

Программа предназначена для работы с материалами ГТИ, ГИС и газового каротажа,
создания многотрековых планшетов, расчёта производных кривых, описания литологии,
выделения интервалов и подготовки мастерлогов.

## Основные возможности

- импорт LAS, CSV, TXT и Excel;
- выбор LAS-кривых по мнемонике, единице, описанию, заполненности и диапазону;
- встроенный справочник Sensors с каноническими мнемониками, псевдонимами и рекомендуемыми диапазонами;
- семантическое распознавание LAS-параметров независимо от порядка колонок, вариантов мнемоник, RU/KK/EN-описаний и единиц;
- автоматическое построение читаемого многотрекового планшета с раздельными шкалами несовместимых параметров;
- синхронная прокрутка и точный видимый интервал планшета по глубине или времени: 1, 5, 10–100 и произвольное значение;
- работа с глубинными, временными и пользовательскими индексами;
- редактирование кривых, заголовков LAS и метаданных;
- расчёт газовых отношений и пользовательских формул;
- работа с компонентами C1–C5, расчётным `TG_CALC` и относительным составом `C1_REL`–`C5_REL`;
- создание литологических колонок, описаний пород и стратиграфии;
- цельная глубинная рабочая форма GeoData с секциями «Геология», «Технология» и «Газовые данные», общей глубинной координатой и синхронной шапкой колонок;
- создание и повторное редактирование литологии жестом `Shift + левая кнопка` и двойным щелчком;
- единый редактор пробы шлама: до четырёх пород с суммой 100%, ЛБА, кальциметрия, rich-text описание, изображения, повторная правка и удаление без дубликатов;
- готовая редактируемая геолого-геохимическая форма Masterlog со шапкой, стратиграфией, бурением, глубиной, шламограммой, ЛБА, кальциметрией, литологией, газом и описанием пород;
- корректная отрисовка LAS-кривых: реальный ноль сохраняется, NULL/NaN создаёт разрыв, линия не соединяется через отсутствующий интервал, а полностью пустой газовый отсчёт не становится ложным нулём;
- создание интерпретаций и глубинных интервалов;
- рисование интервалов и изменение их границ мышью непосредственно на планшете;
- отображение интервалов на планшете и редактирование их свойств;
- Undo/Redo для операций редактирования;
- сохранение проектов без изменения исходных файлов;
- экспорт данных и интервалов в LAS, JSON, CSV и Excel;
- универсальный центр физической печати и экспорта планшетов/графиков в PDF, PNG, JPEG/JPG, TIFF, BMP, WebP и SVG с A4/A3, книжной/альбомной ориентацией, полями, DPI и автоподбором колонок;
- интерфейс на русском, казахском и английском языках с мгновенным переключением без перезапуска.

## Запуск

Требуется Python 3.11 или новее.

### Windows

```powershell
py -3.11 -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -e ".[dev]"
geolog-gasratio-pixler
```

### Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install -e ".[dev]"
geolog-gasratio-pixler
```
