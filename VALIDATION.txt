GEOLOG GASRATIO@Pixler 0.7.10 — Readable parameter labels and explicit pencil save
Date: 2026-07-21

Implemented and verified
- Legacy vendor mnemonics S<number> resolve through Sensors.DB legacy_gid records.
- Verified examples: S300 -> Давление на манифольде; S720 -> Суммарный объем в емкостях; S800 -> Температура на входе; S900 -> Температура раствора на выходе; S50 -> Число ходов 1 насоса.
- Exact LAS descriptions take priority when a vendor reuses an S-code for another channel (for example S800 described as methane remains C1).
- Old layouts/forms where display_name equals the raw mnemonic no longer suppress the readable catalog label.
- Readable names are used in tablet graph headers, curve settings, saved user forms and data exports; original mnemonics remain available in selectors, tooltips and metadata.
- Main-toolbar pencil activation prefers the currently selected editable curve.
- The active pencil track is marked with an orange header and a visible pencil symbol.
- A pencil stroke modifies the in-memory working copy and immediately recalculates existing dependent Gas Ratio/Pixler/custom-formula curves.
- Pencil edit, undo and redo mark the project dirty; disk writing occurs only through explicit Save/Save As.
- The tablet reports “Not saved — press the diskette button” until project save succeeds.
