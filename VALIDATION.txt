GEOLOG GASRATIO@Pixler 0.7.13 — Compact pencil, visible point connection and reliable Undo/Redo

Implemented
- Replaced the oversized diagonal cursor with a compact 26×26 pencil.
- The cursor hotspot is the graphite tip, so the edited value matches the visible contact point.
- Added permanent high-contrast mode buttons:
  - ✎ Freehand / Свободно / Еркін
  - ●—● Connect points / Соединить точки / Нүктелерді қосу
- Kept the existing point workflow: select at least two points, then apply with Connect, Enter or a double-click on the last point.
- Added visible Undo and Redo buttons to the tablet pencil bar.
- Added curve-specific Undo/Redo actions to the track context menu.
- Changed Ctrl+Z and Ctrl+Shift+Z to application-wide shortcuts so plot/combo/table focus cannot swallow them.
- Synchronized pencil history button enabled states with CurveEditHistory.
- Saving clears the dirty marker but intentionally keeps Undo/Redo history; an undo after Save marks the project dirty again.
- Undo and redo continue to recalculate dependent Gas Ratio, Pixler and custom-formula curves in memory.
- Disk persistence still requires explicit Save / diskette action.
- Russian, Kazakh and English resources contain the same 1190 keys.

Required Windows smoke test
1. Open a wide Masterlog tablet and enable the pencil.
2. Confirm the cursor is compact and its tip matches the edited value.
3. Select “Соединить точки”, place two or more points and apply with the button, Enter and double-click.
4. Confirm Undo/Redo buttons and Ctrl+Z/Ctrl+Shift+Z restore both the source curve and recalculated dependants.
