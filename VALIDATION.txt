GEOLOG GASRATIO@Pixler 0.7.12 — Persistent pencil cursor and live value readout

Implemented
- The active tablet plot reasserts the custom pencil cursor on every mouse press, move, release and double-click event.
- The PlotWidget and its viewport both receive the pencil cursor so pyqtgraph/Qt cannot silently restore the default arrow.
- A floating in-plot card follows the pointer and shows:
  - active mnemonic;
  - depth/time coordinate;
  - proposed value under the pencil;
  - previous value of the nearest stored sample.
- The card remains visible while the pointer is stationary.
- The last card position/value is restored after the in-memory curve update and dependent recalculation refresh.
- The card is hidden only when the pointer leaves the target graph, the target changes or pencil mode is disabled.
- Dependent Gas Ratio, Pixler and custom-formula curves still recalculate immediately in memory.
- Disk persistence still occurs only through explicit Save / diskette action.
- Russian, Kazakh and English UI resources contain the same 1182 keys.
