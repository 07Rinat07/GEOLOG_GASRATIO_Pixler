# Formula profiles

Сюда добавляются только подтверждённые профили формул Pixler, Wetness, Balance,
Character, коэффициента флюидности и D-экспоненты.

Каждый профиль должен содержать:

- ID и версию;
- точную формулу;
- перечень входов и единиц;
- источник/методику;
- контрольный численный пример;
- тесты граничных случаев.

Формулы не должны добавляться по памяти или без проверяемого источника.
