# Инкремент 0.7 — Gas, Pixler and DEXP Engine

## Цель

Добавлять расчётные методики только как проверяемые, версионированные профили с provenance.

## План

1. [x] Строгий паспорт формулы: источник, выражение, единицы и версия.
2. [x] Обязательный контрольный численный пример при регистрации.
3. [x] Автоматическое построение dependency graph из профилей.
4. [x] Application-controller выполнения выбранного профиля с mapping и проверкой единиц.
5. [x] UI просмотра паспорта, mapping входных кривых и запуска расчёта.
6. [x] Подтверждённые профили Haworth/Pixler с первичными источниками и контрольными примерами.
7. [x] Профили DEXP/DEXPC с первичными источниками, единицами и проверкой области определения.
8. [x] Расчёт нормального тренда уплотнения с явным интервалом и маской пригодности.
9. [x] Специализированный DEXP/NCT-трек на планшете.
10. [ ] Интеграционные и UI-тесты расчётного сценария.

## Правило безопасности

Профиль без источника, точного выражения, единиц или проходящего контрольного примера не
регистрируется. Непроверенные формулы не добавляются в приложение даже как скрытые defaults.
