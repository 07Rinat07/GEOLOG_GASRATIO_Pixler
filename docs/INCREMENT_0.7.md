# Инкремент 0.7 — Gas, Pixler and DEXP Engine

## Цель

Добавлять расчётные методики только как проверяемые, версионированные профили с provenance.

## План

1. [x] Строгий паспорт формулы: источник, выражение, единицы и версия.
2. [x] Обязательный контрольный численный пример при регистрации.
3. [x] Автоматическое построение dependency graph из профилей.
4. [ ] Application-controller выполнения выбранного профиля.
5. [ ] UI просмотра паспорта и запуска расчёта.
6. [ ] Подтверждённые профили Pixler/Fluid/DEXP после фиксации источников.
7. [ ] Интеграционные и UI-тесты расчётного сценария.

## Правило безопасности

Профиль без источника, точного выражения, единиц или проходящего контрольного примера не
регистрируется. Непроверенные формулы не добавляются в приложение даже как скрытые defaults.
