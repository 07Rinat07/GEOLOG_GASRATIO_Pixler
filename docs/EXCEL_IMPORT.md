# Импорт Excel

Команда «Файл → Импортировать Excel...» открывает XLSX/XLSM только для чтения. В диалоге
можно выбрать лист, номер строки с заголовками и индексную колонку. Предпросмотр ограничен,
а полная книга читается `openpyxl` в режимах `read_only=True` и `data_only=True`: формулы не
исполняются, используются сохранённые в книге результаты.

Заголовки должны быть непустыми и уникальными. Пустые строки пропускаются, пустые ячейки
преобразуются в NULL. Числовые, ISO-8601 и нативные Excel date/time значения проходят через
ту же нормализацию и доменную модель, что CSV/TXT. Оригинальная книга не изменяется.

Поддержка Excel входит в основные зависимости приложения и устанавливается автоматически:

```bash
pip install -e .
```

Старый бинарный `.xls` намеренно не принимается: для него потребуется отдельный изолированный
адаптер. Макросы XLSM не выполняются и не переносятся, поскольку импорт создаёт dataset, а не
новую книгу.
