# Release notes 0.7.24 — тестовый hotfix чёрного планшета

## Исправление

В 0.7.23 полноразмерный прозрачный `QWidget` слоя аннотаций мог некорректно смешиваться с нативными viewport PyQtGraph на Windows и закрывать тело планшета чёрным прямоугольником.

В 0.7.24 слой начинает с пустой нативной области и расширяет её только до реальных границ видимых комментариев, выносок, линий и маркеров. Область дополнительно пересекается с телом графиков ниже шапок.

## Сохранено

- единый ООП-маршрутизатор событий мыши;
- прозрачность слоя для событий мыши;
- редактирование дорожек;
- создание, выбор, перемещение и изменение размера аннотаций;
- совместимость проектов 0.7.23.

## Статус проверки

Модульные и source-level тесты выполняются в контейнере. Настоящий Windows GUI-рендер здесь недоступен из-за отсутствия PySide6 и pyqtgraph, поэтому архив помечен как тестовый.
