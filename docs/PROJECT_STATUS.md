# Статус проекта GEOLOG Gas Ratio & Pixler

## Текущий рабочий модуль

Tablet Engine 2.0 — Rendering Engine.

## Последнее выполненное изменение

Добавлены кэш статической конфигурации дорожек, явная dirty-инвалидация и частичное обновление одной дорожки. Изменение цвета, толщины, сетки, подписи оси или ширины больше не пересоздаёт весь планшет и не затрагивает соседние дорожки.

## Следующая задача

Разделить overlays на независимые слои для annotations, selection и cursor, затем добавить панель диагностики производительности.
