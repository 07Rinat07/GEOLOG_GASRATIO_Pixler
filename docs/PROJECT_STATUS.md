# Статус проекта

Срез: 23 июля 2026 года. Версия пакета: 0.7.28, тестовая сборка.

## Решение о выпуске

Статус релизного gate: **красный**. Сборку нельзя обозначать как stable, пока полный pytest,
Ruff, mypy и обязательный Windows GUI/print smoke-test не завершатся успешно.

## Подтверждённая рабочая основа

- LAS 1.2/2.0, CSV/TXT/Excel и GeoScape/Paradox workflows;
- безопасные копии, таблица LAS, операции над кривыми, Undo/Redo и формулы;
- проект с несколькими datasets и индексами, формат проекта 15;
- синхронный многотрековый планшет, формы, интервалы и литология;
- редактируемый Masterlog, header/forms, PDF и Print Center;
- крупные и промежуточные сетки экрана/печати с сохранением в форме;
- синхронное выделение числового интервала `Shift + ЛКМ`, сводка всех видимых параметров и экспорт XLSX/CSV;
- аннотации, project assets и миграции старых проектов;
- синхронные наборы пользовательских документов RU/KK/EN.

## Технический долг с наибольшим риском

- `tablet/tablet_view.py` — около 339 КБ;
- `ui/main_window.py` — около 254 КБ;
- `printing/masterlog_renderer.py` — около 91 КБ;
- статическая типизация не используется как обязательный проходной критерий;
- старые документы содержали смешение плана и истории, которое теперь вынесено из
  активного статуса.
