# Статус проекта GEOLOG Gas Ratio & Pixler

Добавлен кэш подготовленной геометрии кривых. Для одинаковой кривой, оси, диапазона и бюджета точек повторно используется готовая peak-preserving выборка. Неизменившиеся кривые не получают повторный `setData()`.
