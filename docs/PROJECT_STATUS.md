## Завершено в 0.7.62

- выполнен аудит пользовательской документации и инструкции по всем основным функциям проекта;
- добавлена единая карта возможностей со ссылками на специализированные документы;

- добавлен независимый от Qt аудит локализованных файлов, внутренних ссылок, i18n-ключей,

- корневой README остаётся кратким и не содержит технической истории релизов.
