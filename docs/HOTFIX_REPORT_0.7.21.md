# Hotfix report 0.7.21

## Причина мерцания

Мерцание не было частым автосохранением файла проекта. Геометрия фиксировалась только при отпускании мыши, но после фиксации выполнялся полный `TabletView.refresh_view()`: пересоздавались дорожки, кривые и шапки. Дополнительно прозрачный QWidget повторно применял нативную маску и перерисовывался целиком на каждом событии движения мыши.

## Исправление

- `TabletAnnotationOverlay.mouseMoveEvent` обновляет только объединённый dirty-прямоугольник старого и нового положения;
- во время жеста маска рабочей области остаётся стабильной; одинаковые `QRegion` кэшируются;
- включён `WA_NoSystemBackground`, чтобы ОС не стирала прозрачный фон перед локальной перерисовкой;
- `TabletView.set_canvas_objects()` и `set_image_assets()` синхронизируют только слой аннотаций;
- `MainWindow` не обновляет дерево и весь планшет после изменения геометрии;
- сигнал сохранения геометрии отправляется только при фактическом изменении;
- `DepthAnnotationController.set_geometry()` игнорирует no-op с допуском 0,01 px;
- один drag/resize остаётся одной Undo-командой.

## Совместимость

Схема проекта и формат аннотаций не изменены. Сохранённые выноски, комментарии, изображения, PDF/печать и привязки к глубине/времени/кривой остаются совместимыми.

## Проверка

- `compileall`: успешно;
- 192 целевых dependency-free тест: успешно;
- дополнительный набор изменённых модулей: 24 теста, успешно;
- GUI-проверка Windows/HiDPI не выполнена в контейнере из-за отсутствия PySide6/pyqtgraph.
