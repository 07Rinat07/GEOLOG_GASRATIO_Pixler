# 0.7.16 шығарылым ескертпелері

Нұсқа GeoScape/Paradox DB → бірыңғай `Dataset` моделі → қолданыстағы редактор → жоба/LAS/график/планшет ағынын қосады. Бинарлық қауіпсіз reader, DB/PX/TV/FAM іздеу, тереңдік/уақыт кандидаттары, профильдер, арналар сөздігі, сапаны бақылау, уақыт және тереңдік LAS, TIME → DEPTH, пакеттік конвертер, прогресс және бас тарту іске асырылды.

Тексерілген үлгілер: `BLData.db` — 3488 жазба/70 өріс, `D250.db` — 1739/101. Бастапқы файлдар өзгермеді. Ағымдағы контейнерде `lasio` жоқ, сондықтан жасалған LAS-ты нақты қайта ашу Windows ортасында орындалуы керек.
