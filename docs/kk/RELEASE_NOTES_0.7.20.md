# 0.7.20 шығарылым ескертпелері

## Күн мен уақытты бірыңғай көрсету

- Windows және Linux жүйелерінде бірдей `КК.АА.ЖЖЖЖ СС:ММ:СС` форматтаушысы енгізілді.
- Планшет меңзері, LAS кестесі, қисық терезесі, Paradox алдын ала қарауы, аннотация редакторы және баспа инспекциясы күнтізбелік уақытты енді түсініксіз секунд саны ретінде көрсетпейді.
- `numpy.datetime64`, Unix seconds/milliseconds, Delphi/OLE Automation date және салыстырмалы уақыт қолдау табады.
- Күнтізбелік индекс немесе басталу күні бар болса, толық күн мен уақыт көрсетіледі; бастапқы уақыт жоқ болса, `СС:ММ:СС` қолданылады.
- Пайдаланушыға көрсету жолдарынан ОЖ-ға тәуелді `datetime.fromtimestamp` және `utcfromtimestamp` алынып тасталды.

## F4 тікелей аннотация режимі

1. F4 режимін қосыңыз.
2. «Сілтеме», «Түсініктеме» немесе «Кескін» құралын таңдаңыз.
3. Қажетті жолақтағы нақты тереңдік/уақыт орнын шертіңіз.
4. Нысан бірден таңдалған нүктеде пайда болады.
5. Блокты фоннан/жақтаудан сүйреңіз және сегіз маркермен өлшемін өзгертіңіз.
6. Екі рет шерту, F2, Enter, панель батырмасы немесе контекстік мәзір толық редакторды ашады.

Аннотация қабаты график денесімен шектелді: жақтау, мәтін, сызық және маркерлер жолақ тақырыбы мен параметр жазбаларының үстіне шықпайды. Жоғарғы немесе төменгі шекараға жақын жасалған нысан көрінетін аймақтың ішінде автоматты орналастырылады.

## Үйлесімділік

Жоба форматы мен аннотация схемасы өзгерген жоқ. 0.7.15–0.7.19 жобалары көшірусіз ашылады. Бастапқы LAS/DB файлдары өзгертілмейді.
