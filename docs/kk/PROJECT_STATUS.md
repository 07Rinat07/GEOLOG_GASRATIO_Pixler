# Жоба күйі

Кесім: 2026 жылғы 23 шілде. 0.7.28 нұсқасы, тест жинағы.

## Шығарылым шешімі

Release gate қызыл. Толық pytest өтпейді және Qt жолында авариялық аяқталады; бірінші жеке
қате — `TabletView._annotation_ancestor` әдісінің болмауы. Ruff 6 қатені, mypy 33 файлда
142 қатені көрсетеді. Бұлар түзетіліп, Windows/HiDPI/PDF/принтер smoke матрицасы
орындалмайынша stable күйіне рұқсат жоқ.

## Жұмыс істейтін негіз

- LAS, CSV, TXT, Excel және Paradox үшін қауіпсіз import/edit сценарийлері;
- бірнеше dataset және индексі бар жобалар;
- көп жолақты планшет, аралықтар, литология және пішіндер;
- Masterlog, PDF, Print Center және бапталатын негізгі/аралық торлар;
- `Shift + ЛКМ` аралық таңдауы, барлық көрінетін параметр статистикасы және XLSX/CSV экспорты;
- аннотациялар, project assets және ескі жобаларды көшіру;
- синхронды RU/KK/EN пайдаланушы құжаттары.

Торға арналған 116 және аралық статистикасына арналған 38 нысаналы тест өтті, бірақ олар
толық gate-ті алмастырмайды.

Келесі межелік нүкте: Ruff/mypy/pytest қатесі жоқ, Qt crash жоқ және Windows smoke-test
қол қойылған. Толығырақ: [аудит](PRODUCT_AUDIT_2026.md) және [жоспар](PROJECT_PLAN.md).
