# Жоба күйі

Кесім: 2026 жылғы 23 шілде. 0.7.30 нұсқасы, тест жинағы.

## Шығарылым шешімі

Соңғы толық 0.7.28 baseline жасыл: Ruff өтті, mypy 262 бастапқы файлда 0 қате көрсетті,
толық pytest нәтижесі — 1217 өтті және 10 өткізіліп жіберілді. 0.7.30 үшін `compileall`, wheel
жинағы және session binding, workspace командалары, project controllers пен баспа шекарасын
қамтитын 73 headless/regression/source-integrity тесті сәтті орындалды. Толық gate орнатылған
ортада қайта орындалуы тиіс, себебі ағымдағы контейнерде PySide6, pyqtgraph, lasio, Ruff және
mypy жоқ, ал қолжетімді package index бұл пакеттерді бермейді. Толық gate және
Windows/HiDPI/PDF/физикалық баспа тексерілгенше жинақ тест күйінде қалады.

## Жұмыс істейтін негіз

- LAS, CSV, TXT, Excel және Paradox үшін қауіпсіз import/edit сценарийлері;
- бірнеше dataset және индексі бар жобалар;
- көп жолақты планшет, аралықтар, литология және пішіндер;
- Masterlog, PDF, Print Center және бапталатын негізгі/аралық торлар;
- `Shift + ЛКМ` аралық статистикасы және XLSX/CSV экспорты;
- аннотациялар, project assets және ескі жобаларды көшіру;
- синхронды RU/KK/EN пайдаланушы құжаттары.

Баспа операциялары енді `PrintJobExecutor` арқылы орындалады. `SessionBindingController`
26 контроллерді жаңа project session-ға байланыстырып, Undo/Redo немесе уақытша таңдауды
тазартады. Жоба ағашының әрекеттері `WorkspaceCommandController` арқылы өтеді, сондықтан
`MainWindow` ағаш өңдегіші `current_well_id/current_dataset_id` мәндерін тікелей өзгертпейді.
Келесі кезең — UI кластарының сериализацияланатын жоба моделін тікелей өзгертуіне тыйым салу.
