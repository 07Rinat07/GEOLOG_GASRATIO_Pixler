# Жоба күйі

Кесім: 2026 жылғы 23 шілде. 0.7.29 нұсқасы, тест жинағы.

## Шығарылым шешімі

Соңғы толық 0.7.28 baseline жасыл: Ruff өтті, mypy 262 бастапқы файлда 0 қате көрсетті,
толық pytest нәтижесі — 1217 өтті және 10 өткізіліп жіберілді. 0.7.29 үшін бастапқы код
компиляциясы және import jobs, адаптерлер мен project-session boundary үшін кеңейтілген
регрессиялық жиын орындалды: 105 тест өтті, 3 платформалық тест өткізілді; 0.7.29 wheel пакеті қатесіз жиналды. Толық gate орнатылған ортада қайта орындалуы тиіс, себебі ағымдағы контейнерде
PySide6, pyqtgraph, lasio, Ruff және mypy жоқ, ал package index қолжетімсіз болды.
Толық gate және Windows/HiDPI/PDF/физикалық баспа тексерілгенше жинақ тест күйінде қалады.

## Жұмыс істейтін негіз

- LAS, CSV, TXT, Excel және Paradox үшін қауіпсіз import/edit сценарийлері;
- бірнеше dataset және индексі бар жобалар;
- көп жолақты планшет, аралықтар, литология және пішіндер;
- Masterlog, PDF, Print Center және бапталатын негізгі/аралық торлар;
- `Shift + ЛКМ` аралық таңдауы, көрінетін параметр статистикасы және XLSX/CSV экспорты;
- аннотациялар, project assets және ескі жобаларды көшіру;
- синхронды RU/KK/EN пайдаланушы құжаттары.

`ImportJobController` және `DatasetImportJobExecutor` енді `services/import_jobs.py` ішінде.
Бірыңғай шекара CSV/Excel жоспарларын орындайды, LAS strict/compatible/manual саясатын
қолданады, lossless source/import report сақтайды және Paradox dataset тіркейді. `MainWindow`
тек пайдаланушы таңдауын жинап, нәтижені көрсетеді; қабылданбаған немесе қате файл жартылай
dataset жасамайды. Келесі кезең — print jobs және session binding.
