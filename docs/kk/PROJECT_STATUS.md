# Жоба күйі

Кесім: 2026 жылғы 23 шілде. 0.7.28 нұсқасы, тест жинағы.

## Шығарылым шешімі

Release gate автоматты бөлігі жасыл: Ruff өтті, mypy 260 бастапқы файлда 0 қате көрсетті,
толық pytest нәтижесі — 1205 өтті, 10 өткізіліп жіберілді, аяқталу коды 0.
Windows/HiDPI/PDF/физикалық принтер smoke матрицасы қол қойылғанша жинақ тест күйінде қалады.

## Жұмыс істейтін негіз

- LAS, CSV, TXT, Excel және Paradox үшін қауіпсіз import/edit сценарийлері;
- бірнеше dataset және индексі бар жобалар;
- көп жолақты планшет, аралықтар, литология және пішіндер;
- Masterlog, PDF, Print Center және бапталатын негізгі/аралық торлар;
- `Shift + ЛКМ` аралық таңдауы, барлық көрінетін параметр статистикасы және XLSX/CSV экспорты;
- аннотациялар, project assets және ескі жобаларды көшіру;
- синхронды RU/KK/EN пайдаланушы құжаттары.

Аннотация hit routing қалпына келтірілді, Windows SVG файл құлпы және PySide6 PNG-буфер
үйлесімділігі түзетілді. Планшет панельдері енді тар экранға бейімделеді.

Annotation event router, навигация және толық track lifecycle `TabletView` ішінен бөлінді.
Құру, rollback, қайта реттеу, Undo/Redo және registry тазалау тесттермен тексерілді.
Ортақ grid renderer экран мен баспаға бірдей негізгі/аралық бөліністерді береді, ал тереңдік
жолы сақталған тор баптауларын орындайды. Edit-mode coordinator енді F4 және белсенді
аннотация құралы күйінің жалғыз көзі. `WorkspaceController` деректердің қолжетімділігін және
басты бет/workspace/мақсатты қойынды ауысуларын басқарады. Келесі архитектуралық кезең —
`MainWindow` ішінен import jobs бөліп шығару.
