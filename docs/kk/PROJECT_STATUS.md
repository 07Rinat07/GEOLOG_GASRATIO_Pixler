# Жоба күйі

Кесім: 2026 жылғы 23 шілде. 0.7.31 нұсқасы, тест жинағы.

## Шығарылым шешімі

Соңғы толық 0.7.28 baseline жасыл: Ruff өтті, mypy 262 бастапқы файлда 0 қате көрсетті,
толық pytest нәтижесі — 1217 өтті және 10 өткізіліп жіберілді. 0.7.31 үшін `compileall`, wheel
жинағы және қолжетімді headless regression орындалды: 714 тест өтті, 4 платформалық сценарий
өткізіліп жіберілді. Толық pytest жинауы 95 Qt/LAS тәуелді модульде тоқтайды, себебі контейнерде
PySide6, pyqtgraph және lasio жоқ; Ruff пен mypy де қолжетімсіз. Толық gate және
Windows/HiDPI/PDF/физикалық баспа тексерілгенше жинақ тест күйінде қалады.

## Жұмыс істейтін негіз

- LAS, CSV, TXT, Excel және Paradox үшін қауіпсіз import/edit сценарийлері;
- бірнеше dataset және индексі бар жобалар;
- көп жолақты планшет, аралықтар, литология және пішіндер;
- Masterlog, PDF, Print Center және бапталатын негізгі/аралық торлар;
- `Shift + ЛКМ` аралық статистикасы және XLSX/CSV экспорты;
- аннотациялар, project assets және ескі жобаларды көшіру;
- синхронды RU/KK/EN пайдаланушы құжаттары.

UI-дың күй өзгертетін әрекеттері енді controller/service шекаралары арқылы өтеді. Планшет
жолақтарының енін/ретін, тік индексті және көрінетін аралықты өзгерту бөлек headless mutation
шекарасына берілді; `MainWindow` сериализацияланатын коллекциялар мен `dirty` күйін тікелей
өзгертпейді. Тоқтатылған merge немесе external-LAS export уақытша dataset-ті транзакциялық
түрде жойып, бастапқы таңдау мен `dirty` күйін қалпына келтіреді. `SessionBindingController`
енді 27 контроллерді қайта байланыстырады.
