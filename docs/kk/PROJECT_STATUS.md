- мөлдір кесілген сурет немесе фоны бар бастапқы нұсқа таңдалады;
- енгізу алдында жолақ, параметр/тереңдікке байлау, нақты тереңдік және бастапқы өлшем беріледі;
- нысанды сол жақ батырмамен жылжытып, сегіз маркермен масштабтауға болады;
- белгі жобаның ішкі қоймасында catalog ID және фон режимімен бірге сақталады;
- қолданыстағы Undo/Redo, жобаны сақтау, PDF және баспа механизмі пайдаланылады;
- құжаттама мен нұсқаулықтар орыс, қазақ және ағылшын тілдерінде синхрондалды;
- түбірлік README-де техникалық релиз тарихы жоқ.
