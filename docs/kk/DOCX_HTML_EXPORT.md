# Таңдалған аралықты DOCX және HTML пішімдеріне экспорттау

0.7.40 нұсқасы **Файл** мәзіріне екі жаңа пішім қосты:

- **Таңдалған деректерді DOCX-ке экспорттау** — Microsoft Word-та өңделетін құжат;
- **Таңдалған деректерді HTML-ге экспорттау** — браузерге және баспаға арналған бір автономды файл.

## Экспорттау тәртібі

1. Dataset ашыңыз.
2. Графикте немесе кестеде тереңдік аралығын таңдаңыз.
3. Қажетті қисықтарды белгілеңіз.
4. **Файл** мәзірінен DOCX немесе HTML таңдаңыз.
5. Файл атауын көрсетіп, қажет болса ауыстыруды растаңыз.

Екі пішім де CSV және XLSX қолданатын дәл сол resolved interval-ды пайдаланады. Бағдарлама
viewport-ты қайта оқымайды және таңдалған индексті басқа индексімен жасырын ауыстырмайды.

## Құжат құрамы

- dataset және есеп атауы;
- нақты шекаралар мен есептер саны;
- ReportDefinition SHA-256;
- арналардың coverage кестесі;
- таңдалған аралық мәндерінің кестесі.

Белгілер:

- `0` — нақты өлшенген нөл;
- `—` — өткізіп алынған есеп;
- `#N/A` — сұралған арна dataset ішінде жоқ.

HTML интернетті қажет етпейді және сыртқы script немесе style жүктемейді. DOCX ішінде
макростар мен сыртқы объектілер жоқ.

Файлдың жанында `*.report-passport.json` жасалады. Passport дайын DOCX/HTML файлының
өлшемін, MIME түрін және SHA-256 мәнін сақтайды. Экспорттан кейін файл өзгерсе, passport
тексеру оны анықтайды.
