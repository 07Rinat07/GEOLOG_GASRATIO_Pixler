# Delphi SKF пішіндерін импорттау

0.7.14 нұсқасы Delphi компоненттерінің бинарлық ағыны бар ескі `.skf` файлдарына арналған қауіпсіз импорттағышты қосады.

## Нәтиже

Бір SKF файлы мына нысандарға түрленеді:

- планшетке арналған өңделетін `FormDocument`;
- жобадағы HeaderTemplate ішкі үлгісі болып табылатын байланыстырылған `MasterlogTemplate`;
- ағыннан табылған BMP, PNG немесе JPEG суреттері;
- бастапқы файл атауы, өлшемі, SHA-256, түбірлік класс, компонент саны және ескертулер бар есеп.

Пішін `forms/depth` немесе `forms/time` пайдаланушы кітапханасына сақталады. Байланыстырылған тақырып Конструктордың пайдаланушы баспа үлгілерінде көрінеді.

## Қолдану

**Пішіндер → Пішіндер кітапханасы → Импорт және экспорт → SKF импорттау** командасын таңдаңыз. Сол команда **Конструктор → Планшет пішіндері** бөлімінде де бар.

Импорттағыш танылған геометрияны, атауларды, мәтін бағытын, мнемоникаларды, өлшем бірліктерін, түстерді, сызық стильдерін, шкалаларды, min/max аралығын, торды, тақырып өрістерін, сызықтар мен суреттерді тасымалдайды. Белгісіз Delphi компоненттері эвристикалық түрде сәйкестендіріледі немесе ескертумен өткізіледі.

## Қауіпсіздік

Delphi кластары жүктелмейді және оқиға өңдегіштері орындалмайды. Ағын файл өлшемі, бинарлық қасиеттер, тереңдік және компонент саны шектелген бейтарап ағашқа талданады.

## Диагностика

```powershell
python tools\inspect_skf.py form.skf --dump-json skf-tree.json
python tools\inspect_skf.py form.skf --convert-dir converted
```

Синтетикалық Delphi ағыны тексерілді. Ескі бағдарламаның арнайы компоненттерін дәл сәйкестендіру үшін пайдаланушының нақты SKF үлгілері қажет.
