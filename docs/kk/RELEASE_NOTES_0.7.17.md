# 0.7.17 шығарылым ескертпелері

## Шығарылым мақсаты

Бұл hotfix 0.7.16 нұсқасын қолмен тексеру кезінде табылған кәсіби аннотация қабатының нақты ақауларын түзетеді. Жоба пішімі мен сақталған аннотация моделі өзгермейді.

## Түзетілді

- F4 панеліндегі «Выноска», «Түсініктеме» және «Сурет» батырмалары редакторды ашады;
- панельден және контекстік мәзірден жасау «Сақтау/Бас тарту» терезесін қолданады;
- мәтін, стиль, ығысу, ен және биіктік сақтау алдында өңделеді;
- бар блок тінтуірмен жылжиды, төменгі оң жақ маркермен өлшемін өзгертеді және екі рет басумен ашылады;
- объект үстіндегі оң жақ батырма жолақ мәзірінің орнына объект мәзірін ашады;
- жаңа выноскада бекіту нүктесі мен мәтін блогы арасында анық сызық бар;
- GeoScape 0,2 м стандарты DB нақты қадамынан бөлек көрсетіледі. Берілген BLData үшін анық ресэмплингке дейін LAS `STEP=0.4` сақталады.

## Үйлесімділік

0.7.15/0.7.16 жобалары көшірусіз ашылады. Бастапқы LAS/DB өзгермейді.
