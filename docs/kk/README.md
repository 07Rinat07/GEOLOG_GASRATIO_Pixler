# Пайдаланушы нұсқаулығы

GEOLOG GASRATIO@Pixler — бұрғылау, газ каротажы және LAS деректерінің редакторы.

## Тіл

Алғашқы іске қосылғанда Русский, Қазақша немесе English тілін таңдаңыз. Кейін тілді
«Тіл / Language / Язык» мәзірінен өзгертуге болады. Барлық терезелерді бірдей жаңарту үшін
қолданбаны қайта іске қосыңыз.

## Импорт

«Файл → Деректерді импорттау...» (`Ctrl+I`) командасын қолданып, LAS, CSV/TXT немесе Excel
түрін таңдаңыз. Бастапқы файлдар өзгертілмейді. CSV/TXT және Excel шеберлері индекс,
DATE/TIME, уақыт белдеуі, алдын ала қарау және тексеру параметрлерін таңдалған тілде көрсетеді.

## Кестені өңдеу

LAS кестесі мән енгізуді, аралықты тұрақты немесе кездейсоқ мәндермен толтыруды, көшіру/кірістіруді
және Болдырмау/Қайталауды қолдайды. Барлық командалар таңдалған тілде көрсетіледі.

## LAS экспорттау

Экспорттау диалогында LAS 1.2/2.0, WRAP, NULL, дәлдік және пайдаланушы бөлімдерін сақтау
параметрлері таңдалады. Ескертулер мен файлды ауыстыру растауы таңдалған тілде көрсетіледі.

## Есептеу формулалары

Formula Profiles терезесі паспортты, өрнекті, шығысты, дереккөзді және кіріс қисықтарының
сәйкестігін көрсетеді. DEXP атауы мен сипаттамасы RU/KK/EN тілдерінде қолжетімді; формула мен
өлшем бірліктері аударылмайды.

Қолданыстағы толық инженерлік құжаттама `docs` каталогында сақталады және RU/KK/EN
пайдаланушы нұсқаулықтарына кезең-кезеңімен көшіріледі.
