# Пайдаланушы нұсқаулығы

GEOLOG GASRATIO@Pixler — бұрғылау, газ каротажы және LAS деректерінің редакторы.

## Тіл

Алғашқы іске қосылғанда Русский, Қазақша немесе English тілін таңдаңыз. Кейін тілді
«Тіл / Language / Язык» мәзірінен өзгертуге болады. Барлық терезелерді бірдей жаңарту үшін
қолданбаны қайта іске қосыңыз.

## Импорт

«Файл → Деректерді импорттау...» (`Ctrl+I`) командасын қолданып, LAS, CSV/TXT немесе Excel
түрін таңдаңыз. Бастапқы файлдар өзгертілмейді.

Қолданыстағы толық инженерлік құжаттама `docs` каталогында сақталады және RU/KK/EN
пайдаланушы нұсқаулықтарына кезең-кезеңімен көшіріледі.
