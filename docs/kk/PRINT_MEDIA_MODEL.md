# Баспа пішімдері мен масштабы

0.7.38 нұсқасы preview, PDF, бет-бет экспорт және физикалық принтер үшін бір модель қолданады.

## Пішімдер

- A4 және A3;
- 25–5000 мм custom парақ;
- ені берілетін және сегмент ұзындығы автоматты түрде 5000 мм-ге дейін есептелетін roll.

## Fit және 100%

**Fit** пішінді бір көлденең бетке сыйғызады. **100%** бағандардың бастапқы енін 96 логикалық
pixel/inch қатынасымен сақтайды және пішін қағаздан кең болса continuation беттерін жасайды.
Continuation overlap миллиметрмен беріледі.

Тік аралық бөлек таңдалады: current, full, custom немесе selection. Сондықтан бір тік аралықта
бірнеше көлденең continuation болуы мүмкін.

## Физикалық принтер

Жүйелік диалогтан кейін бағдарлама media support, custom/roll шектері, minimum margins, printable
area, DPI және таңдалған page range тексереді. Error баспаны тоқтатады; nearest DPI және көп беттік
paper feed warning журналға жазылады.

Құрылғыны тексеру үшін `tools/physical_print_gate.py` қолданыңыз. Нақты test print тек
`--print-test` параметрімен жіберіледі.

Толық инженерлік contract: [Ортақ баспа моделі](../PRINT_MEDIA_MODEL.md).
