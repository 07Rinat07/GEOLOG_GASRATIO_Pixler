# 0.7.62 шығарылым ескертпелері — үш тілдегі құжаттама аудиті

0.7.62 нұсқасы құжаттама мен автоматты сапа тексеруін жақсартады. Project format v20,
form schema v6 және tablet layout v16 өзгерген жоқ.

## Орындалды

- толық нұсқаулықтарға сілтемелері бар өзекті пайдаланушы функциялары картасы қосылды;
- деректер, пішіндер, аралықтар, аннотациялар және белгілердің жоба арқылы бірыңғай сақталуы түсіндірілді;
- белгі нұсқаулығына сақтау, жабу, қайта ашу, Undo/Redo, жою, құлыптау, баспа жалаушасы,
  PDF және суретті жобаның ішкі қоймасында сақтау тәртібі қосылды;
- пайдаланушы нұсқаулығы, күй, жоспар, release notes және навигация синхрондалды;
- RU/KK/EN құжаттар жиынының теңдігін, ішкі сілтемелерді, локализация кілттерін, пакет нұсқасын
  және сақтау/қайта ашу сценарийлерін тексеретін автоматты аудит қосылды;
- құжаттама келісімшартына regression-тесттер қосылды;
- түбірлік README қысқа күйінде қалды және түзетулер тарихымен толтырылмады.
