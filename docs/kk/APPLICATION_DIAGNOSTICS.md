# Қолданба диагностикасы

0.7.51 нұсқасынан бастап GEOLOG GASRATIO@Pixler тұрақты айналмалы журнал жүргізеді.

**«Анықтама → Журналдар бумасын ашу»** командасы каталогты ашады. Ағымдағы файл `geolog.log`,
ал алдыңғы журналдар нөмірленген ротациялар ретінде сақталады. `geolog-crash.log` faulthandler
және авариялық аяқталу мәліметтеріне арналған.

Мәселе туындағанда **«Анықтама → Диагностика пакетін жинау…»** командасын пайдаланыңыз. ZIP:

- ағымдағы және бірнеше алдыңғы журналды;
- қолданба, Python және операциялық жүйе нұсқаларын;
- dataset идентификаторы, тректер саны, қарындаш және пішін транзакциясының күйі сияқты қауіпсіз
  runtime-контексті қамтиды.

LAS мәндері, жоба assets, сақталған пішіндер және жоба файлдары пакетке қосылмайды. Қалыпты
журнал хабарларында файл жолдары мен жоба/dataset атаулары болуы мүмкін, сондықтан пакетті сыртқа
жібермес бұрын тексеріңіз.

Мәселені қайталау үшін қате әрекетті орындап, бірден диагностика пакетін құрыңыз және ZIP-пен
бірге басылған батырмалардың қысқаша ретін жіберіңіз.
