# 0.7.19 шығарылым ескертпелері

## Аннотациялардың тереңдікпен және уақытпен синхрондалуы түзетілді

- Выноска мен түсініктеме көрінетін ось диапазоны өзгерген сайын экрандық байлау нүктесін қайта есептейді.
- Тінтуір дөңгелегі, тік айналдыру жолағы, панорамалау, мәнге өту және масштабтау аннотацияны тиісті тереңдікпен немесе уақытпен бірге жылжытады.
- Мәтін блогы дерек нүктесіне қатысты пайдаланушы орнатқан ығысуын, енін, биіктігін және безендірілуін сақтайды.
- Байланған тереңдік көрінетін аралықтан шыққанда, аннотация график үстінде қозғалмай қалмай, деректермен бірге экраннан шығады.
- Қисыққа байланған аннотация үшін X параметр мәні бойынша, ал Y белсенді тереңдік/уақыт осі бойынша қайта есептеледі.
- Позицияны жаңарту мәтінді, стильді, өлшемді және сақталған ығысуларды өзгертпейді.
- Навигация кезінде графикалық көмекші объектілер қайта пайдаланылады және әр дөңгелек қадамында қайта жасалмайды.

## Үйлесімділік

Жоба пішімі мен `annotation` объектілерінің схемасы өзгерген жоқ. 0.7.15–0.7.18 нұсқаларында жасалған аннотациялар көшірусіз ашылады.

## Тексеру

Регрессиялық тексерулер навигацияның екі жолын да қамтиды: планшеттің ортақ камерасы және pyqtgraph `ViewBox` диапазонының тікелей өзгеруі.
