# Tablet Engine 2.0 — навигация

Планшет барлық жолақтар және тереңдік/уақыт осьтері үшін бір камераны қолданады.

Басқару:

- дөңгелек — айналдыру;
- `Ctrl+дөңгелек` — курсор мәніне қатысты масштаб;
- ортаңғы батырма немесе `Space + сол батырма` — қолмен жылжыту;
- `Home`/`End` — басы/соңы;
- `PageUp`/`PageDown` — бір экранға жуық;
- `↑`/`↓` — кіші қадам;
- айналдыру жолағы және мәнге өту барлық колонкамен синхронды.

Камера терезені LAS деректерінің нақты ауқымымен шектейді.
