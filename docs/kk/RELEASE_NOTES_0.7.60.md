# GEOLOG GASRATIO@Pixler 0.7.60

Интервал статистикасының панелі сыртқы қалқымалы `QDockWidget` терезесінен планшет ішіндегі
overlay-компонентке ауыстырылды. Панель форманың енін кішірейтпейді, экран шегінен шықпайды
және терезе өлшемі өзгергенде пайдаланушы қойған орнын сақтайды. Панель жабылғанда немесе
форма ауысқанда интервал таңдалуы мен статистика толық тазаланады.

Geometry, contract және Qt regression-тесттері қосылды. Түбірлік README тек жоба сипаттамасын,
іске қосуды және құжаттама сілтемелерін қамтиды; техникалық тарих `docs` ішінде сақталады.

Project format v20, form schema v6 және tablet layout v16 өзгерген жоқ.
Контейнердегі тексеру: 19 focused test өтті; қолжетімді headless regression нәтижесі — 1094 passed, 4 skipped, 3 deselected. Qt сценарийлері үшін PySide6 қажет.
