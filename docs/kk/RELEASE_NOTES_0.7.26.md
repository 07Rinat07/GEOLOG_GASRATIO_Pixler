# 0.7.26 шығарылым жазбалары — DB → LAS типтелген импорт жоспарын түзету

## Түзетілген қате

Windows жүйесінде `QComboBox.currentData()` кейбір `StrEnum` мәндерін кәдімгі жол ретінде қайтара алады. Сондықтан пакеттік түрлендіргіштен жасалған қолмен баптау жоспары импорттаушыға `classification="time_with_depth"` немесе `duplicate_depth_policy="keep_all"` жолдары түрінде жетті. Метадеректерді жазу кезінде код `.value` қасиетіне жүгініп, операция мына қатемен аяқталды:

```text
'str' object has no attribute 'value'
```

Енді `ParadoxImportPlan` — жалғыз типтелген шекара. Жоспар жасалған кезде:

- Qt және JSON жолдары `DatasetClassification` және `DuplicateDepthPolicy` түрлеріне түрлендіріледі;
- `active_role`, NULL, тіл, индекс өрістері және арна сәйкестіктері тексеріледі;
- қолдау көрсетілмейтін мәндер оқу немесе экспорт басталғанға дейін қабылданбайды;
- ішкі импорттаушы тек қалыпқа келтірілген жоспармен жұмыс істейді.

Қолмен баптау терезесі де жоспар жасамас бұрын Qt мәндерін айқын түрлендіреді.

## Пакеттік операция диагностикасы

Қате хабарында енді операция кезеңі көрсетіледі:

- DB оқу;
- арналарды талдау;
- импорт жоспарын дайындау;
- деректер жиынын жасау;
- LAS жазу;
- LAS файлын қайта ашу.

## 0,4 м нақты қадам және 0,2 м стандарт

`.value` қатесі тереңдік қадамына байланысты емес. LAS 2.0 форматы міндетті түрде 0,2 м қадамды талап етпейді. Егер бастапқы DB жолдары нақты 0,4 м сайын жазылса, экспорттағы `STEP` 0,4 м болып қалуы тиіс. GeoScape серверінің 0,2 м стандарты метадеректерде бөлек сақталады.

0,2 м торын алу үшін LAS редакторында ресэмплингті айқын орындап, интерполяция әдісін таңдау қажет. Пакеттік түрлендіргіш аралық жолдарды жасырын жасамайды және нақты `STEP` мәнін алмастырмайды.

## Үйлесімділік

- бастапқы DB/PX/TV/FAM файлдары тек оқу режимінде ашылады;
- жоба және аннотация схемалары өзгерген жоқ;
- 0.7.16–0.7.25 профильдері жүктеле береді;
- ескі және жаңа профильдердің жолдық мәндері бірдей қалыпқа келтіріледі;
- нақты Windows `DB → LAS → қайта ашу` сынағына дейін бұл сынақ құрастыруы болып қалады.
