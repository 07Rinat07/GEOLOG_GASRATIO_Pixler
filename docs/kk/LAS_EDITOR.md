# LAS редакторы

Нұсқа: 0.7.6

Негізгі интерфейсте жеке **«Редактор»** бөлімі және көрінетін **«LAS редакторы»**
түймесі (`Ctrl+Alt+E`) бар. Ол LAS жасау, кестені өңдеу, тереңдікті түзету,
қадамды өзгерту, қисықтарды кірістіру және файлдарды біріктіруді бір жерге жинайды.

## Бастапқы файлдардың қауіпсіздігі

Редактор бастапқы LAS файлдарын қайта жазбайды. Түзету, қайта дискреттеу,
кірістіру және біріктіру жеке dataset және жаңа `.las` файлын жасайды. Дереккөз
жолы, кодировкасы, SHA-256 және операция туралы ақпарат metadata/manifest ішінде қалады.

## Операциялар

- **Жаңа LAS жасау** — аралық, қадам, MD/TVD/TVDSS индексі, LAS 1.2/2.0 және NULL.
- **LAS ашу** — файлдарды бөлек ұңғыма жұмыс кеңістіктеріне импорттау.
- **Кестені өңдеу** — мәндерді, тақырыптарды және қисықтарды жаңа көшірме экспортталғанша өзгерту.
- **Кемитін тереңдікті түзету** — тереңдік, индекстер және барлық қисықтарды бірге аудару.
- **Тереңдік қадамын өзгерту** — 0,2; 0,5; 1 м немесе пайдаланушы қадамымен жаңа көшірме.
- **LAS деректерін кірістіру** — сыртқы файл қисықтарын таңдап, тереңдік бойынша қосу.
- **LAS біріктіру** — қабаттасуда ағымдағы мәнді сақтау, дереккөзге басымдық беру немесе екі нұсқаны қалдыру.
- **Ағымдағыны жаңа LAS ретінде сақтау** — жұмыс dataset-ін экспорттау.

## Стандартты емес файлдар

Кемитін тереңдік, теріс `STEP`, CP866/Windows-1251, қайталанатын мнемоникалар және
кириллица қолданылады. Бастапқы атаулар сақталады, ал шығысқа қауіпсіз атаулар
ұсынылады: `GK:1 → GK_1`, `КС, ННК/ДСР → KS_NNK_DSR`.
