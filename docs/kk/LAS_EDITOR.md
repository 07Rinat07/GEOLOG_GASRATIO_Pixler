# LAS редакторы

Функция нұсқасы: 0.7.6. **«Редактор»** бөлімі және **«LAS редакторы»** (`Ctrl+Alt+E`)
командасы LAS жасау, қолмен өңдеу, тереңдікті түзету, қадамды өзгерту, қисықтарды кірістіру және
файлдарды біріктіру әрекеттерін бір жерге жинайды.

## Бастапқы файлдардың қауіпсіздігі

Редактор бастапқы LAS файлын қайта жазбайды. Түзету, қайта дискреттеу, кірістіру және біріктіру
жаңа тәуелсіз dataset пен жаңа `.las` файлын жасайды. Бастапқы жол, кодировка, SHA-256 және
орындалған операция нәтиженің metadata немесе manifest бөлімінде сақталады.

Кесте өзгерістері алдымен жобаның жұмыс dataset-інде болады. **Ctrl+S** жобаны сақтайды, бірақ
жаңа LAS файлын жасамайды. Басқа бағдарламаға беру үшін жаңа LAS көшірмесін экспорттаңыз.

## Негізгі операциялар

- **Жаңа LAS жасау** — аралық, қадам, MD/TVD/TVDSS индексі, LAS 1.2/2.0 және NULL.
- **LAS ашу** — бір немесе бірнеше файлды бөлек жұмыс ұңғымаларына импорттау.
- **Кестені өңдеу** — мәндер, тақырыптар және қисықтар, Undo/Redo қолдауымен.
- **Кемитін тереңдікті түзету** — тереңдік, индекстер және барлық қисықтарды бірге аудару.
- **Тереңдік қадамын өзгерту** — 0,2; 0,5; 1 м немесе басқа оң қадаммен жаңа көшірме.
- **LAS деректерін кірістіру** — сыртқы файл қисықтарын таңдап, тереңдік бойынша сәйкестендіру.
- **LAS біріктіру** — қабаттасу саясаты көрсетілген тереңдік учаскелерін біріктіру.
- **Ағымдағыны жаңа LAS ретінде сақтау** — жұмыс көшірмесін бөлек файлға экспорттау.

## Стандартты емес және ескі LAS

Кемитін тереңдік, теріс `STEP`, UTF-8, Windows-1251 және CP866, қайталанатын мнемоникалар және
кириллица қолдау табады. Бастапқы атаулар диагностика үшін сақталады, ал шығыс LAS үшін
тасымалданатын атаулар ұсынылады: `GK:1 → GK_1`, `КС, ННК/ДСР → KS_NNK_DSR`.
Экспорт алдында ұсынылған атауды өзгертуге болады.

Қадамды өзгерту кезіндегі интерполяция NULL және үлкен үзілістерді қоспайды. Растамас бұрын
аралықты, жол санын, өлшем бірліктерін, NULL және Import Review ескертулерін тексеріңіз.

## Қисықтарды кірістіру

1. LAS қабылдағышын ашыңыз.
2. **Редактор → LAS деректерін кірістіру** командасын таңдаңыз.
3. Сыртқы LAS файлын таңдаңыз.
4. Бастапқы және шығыс мнемоникаларды, өлшем бірліктері мен сипаттамаларды тексеріңіз.
5. Қажет емес қисықтардың жалаушасын алып тастаңыз.
6. Жаңа LAS көшірмесінің жолын көрсетіңіз.
7. **«Кірістіру және көшірмені сақтау»** батырмасын басыңыз.

Кемитін сыртқы файл тек жадта аударылады. Мәндер қабылдағыш торына ортақ тереңдік аралығында
көшіріледі; қабаттасудан тыс жерде NULL жазылады. Бастапқы файлдар өзгермейді.

## LAS файлдарын біріктіру

1. Екі LAS файлын да ашыңыз.
2. Жоба ағашынан қабылдағыш dataset-ті таңдаңыз.
3. **Редактор → LAS біріктіру** командасын іске қосыңыз.
4. Екінші dataset пен қабаттасу саясатын таңдаңыз:
   - ағымдағы мәндерді сақтау;
   - дереккөз мәндеріне басымдық беру;
   - екі қисықты әртүрлі мнемоникамен қалдыру.
5. Жаңа файл атауын көрсетіп, операцияны растаңыз.

Екі бастапқы файл өзгермейді. Нәтижеден басталу/аяқталу тереңдігін, қабаттасуды, NULL,
өлшем бірліктерін және тақырыптарды тексеріңіз.

## Сақтау және қайта ашып тексеру

- **Ctrl+S** жұмыс моделін жоба ішінде сақтайды.
- **LAS экспорттау** бөлек тасымалданатын файл жасайды.
- Жобаны сақтамай жабу соңғы `Ctrl+S` кейінгі кесте өзгерістерін жояды.
- Маңызды өңдеуден кейін жобаны сақтап, LAS көшірмесін экспорттап, екі нәтижені де қайта ашып,
  жол санын, индексті және негізгі қисықтарды тексеріңіз.
