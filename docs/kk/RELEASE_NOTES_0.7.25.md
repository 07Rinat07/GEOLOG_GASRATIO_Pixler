# 0.7.25 шығарылым жазбалары — DB пакеттік баптауы және қауіпсіз планшет рендері

Күйі: Windows-та міндетті визуалды тексеріс өткенге дейін **сынақ құрастыруы**.

## DB → LAS пакеттік түрлендіру

- анық емес индекс енді жай қате болып саналмайды;
- жол **«Баптау қажет»** күйін алады;
- **«Таңдалған DB-ны баптау…»** батырмасы стандартты GeoScape/Paradox терезесін ашады;
- пайдаланушы бөлек құжат ашпай тереңдік, уақыт, арналар және бірліктерді таңдайды;
- таңдалған жоспар ағымдағы сеанста сақталып, depth және time экспортына қолданылады;
- баптаудан кейін тек таңдалған файлды қайта іске қосу ұсынылады;
- нақты қате, бар файлды өткізіп жіберу және баптау сұрауы бөлек көрсетіледі.

## Планшет және аннотациялар

- Windows-та PyQtGraph аймағын қара тіктөртбұрышпен жабуы мүмкін толық өлшемді мөлдір `QWidget` жойылды;
- `TabletAnnotationOverlay` енді жасырын геометрия/сигнал менеджері және өзі ештеңе салмайды;
- әр көрінетін аннотация мөлдір `QPixmap` бар жеке шағын `QLabel`-спрайтпен салынады;
- спрайт нысанның нақты шекарасымен және тақырыптан төмен график денесімен шектеледі;
- планшеттің бос аймақтарының үстінде аннотация терезесі болмайды;
- тінтуір маршрутизациясы, жолақ редакторы, Undo/Redo және баспа рендері сақталды.

## Тексеру

Контейнерде компиляция, модульдік/source-contract тесттер, локализация және нақты Paradox үлгілері тексерілді. Нақты `PySide6 + pyqtgraph` Windows рендері қолжетімсіз, сондықтан құрастыру stable деп жарияланбайды.
