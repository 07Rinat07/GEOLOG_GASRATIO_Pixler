# 0.7.24 шығарылым жазбалары — қара планшетке арналған сынақ hotfix

## Түзету

0.7.23 нұсқасында аннотация қабатының толық өлшемді мөлдір `QWidget` элементі Windows жүйесінде PyQtGraph нативті viewport аймақтарымен дұрыс араласпай, планшет денесін қара тіктөртбұрышпен жабуы мүмкін еді.

0.7.24 нұсқасында қабат бос нативті аймақтан басталады және тек көрінетін түсініктеме, сілтеме, сызық және маркер шекараларына дейін кеңейеді. Аймақ тақырыптардан төмен график денесімен қиылысады.

## Сақталған мүмкіндіктер

- тінтуір оқиғаларының бірыңғай ООП-маршрутизаторы;
- қабаттың тінтуір оқиғаларына мөлдірлігі;
- жолақтарды өңдеу;
- аннотацияларды жасау, таңдау, жылжыту және өлшемін өзгерту;
- 0.7.23 жобаларымен үйлесімділік.

## Тексеру мәртебесі

Контейнерде модульдік және source-level тесттер орындалады. PySide6 және pyqtgraph болмағандықтан нақты Windows GUI рендері тексерілмеді, сондықтан архив сынақ жинағы болып белгіленген.
