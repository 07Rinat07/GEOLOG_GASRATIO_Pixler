# Пішіндер, тақырыптар және баспаға арналған әмбебап конструктор

Жаңартылған күні: 21 шілде 2026
Іске асыру нұсқасы: 0.7.1

## Мақсаты

**«Конструктор»** мәзірі планшет пішіндерін, баспа Masterlog пішіндерін, WYSIWYG
тақырыптарын, бағандарды, бет профильдерін, литотиптерді, тереңдік белгілерін, алдын ала
қарауды және preflight тексеруін бір жерге біріктіреді. Жаңа үйлеспейтін жүйе жасалмайды:
қолданыстағы Masterlog моделі мен renderer пайдаланылады.

## Келесі басымдық

1. Form Manager және тереңдік навигациясының Windows smoke-test-і.
2. Сызғыш, бағыттаушы, бұғаттау және Undo/Redo.
3. Preview ішінде белгілерді тікелей жылжыту.
4. Белгілер аңызы және кеңейтілген preflight.
5. Экрандық және баспа үлгілерін нақты profile ID арқылы байланыстыру.

## Конструктормен байланысты 0.7.1 срезі

Стратиграфиялық блоктарға сақталатын `text_orientation` және `text_position` қасиеттері
қосылды. Олар аралық интерфейсінде өңделеді және экрандық пен баспа renderer-де бірдей
қолданылады. Бұл конструктордың бірыңғай preview талабына кіреді.
