# GEOLOG GASRATIO@Pixler 0.7.46 — LAS импортын қалпына келтіру және диагностика

2026 жылғы 24 шілдедегі жедел тесттік жинақ. Project format **v20** болып қалады.

## Түзетілген регрессия

0.7.45 инженерлік тор overlay-ы PySide6 ішіне типтелген `Qt.MouseButton.NoButton` орнына
жай `0` санын берген. Жаңа PySide6 нұсқаларында LAS сәтті оқылып, review расталып, dataset
тіркелгеннен кейін бірінші планшет кадрын салу кезінде exception пайда болып, жұмыс аймағы
қара болып қалуы мүмкін еді.

0.7.46 типтелген жалаушаны қолданады, overlay істемесе axis-grid fallback қосады және
планшет/қисықтар көрсетілмесе импортталған dataset-ті қауіпсіз кесте режимінде ашады.
Import Review ескертулері импортты бұғаттамайды, ал күтпеген қате әр файл шекарасында ұсталады.

## Әрекетке жарамды диагностика

Импорт `read`, `parse`, `policy`, `review`, `register`, `present` кезеңдері бойынша бір есеп
жинайды. Әр мәселе severity, тұрақты code, source, түсініктеме, түзету әрекеті, context,
exception type және traceback сақтайды. Есепті көшіруге/сақтауға болады; бұғаттайтын қате
кезінде техникалық көшірме application data каталогына атомарлы жазылады.
