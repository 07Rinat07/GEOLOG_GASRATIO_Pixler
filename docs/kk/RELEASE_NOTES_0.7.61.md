# GEOLOG GASRATIO@Pixler 0.7.61

## Графикке белгілер енгізу

Пішінді өңдеу **F4** режиміне жеке **«Белгі енгізу»** командасы қосылды. Терезеде нобайлары,
қазақша атаулары, санаттары және іздеуі бар кірістірілген белгілер анықтамалығы көрсетіледі.
Пайдаланушы мыналарды таңдайды:

- артық фоны кесілген мөлдір нұсқа немесе фоны сақталған бастапқы нұсқа;
- график жолағы;
- қисыққа байлайтын параметр немесе тек тереңдікке байлау;
- нақты тереңдік және бастапқы өлшем.

Енгізілген белгі автоматты түрде таңдалады. Оны тінтуірдің сол жақ батырмасымен дәл орнына
жылжытуға және сегіз бұрыштық/бүйірлік маркер арқылы өлшемін өзгертуге болады. Сол әрекет графиктің
контекстік мәзірінде де бар.

Белгі жобаның ішкі ресурстар қоймасына көшіріледі және анықтамалық идентификаторын әрі фон режимін
сақтайды. Бастапқы BMP/PNG файлына сыртқы жол қажет емес. Нысан Undo/Redo-ға қатысады және экран,
PDF пен баспа үшін бірдей қолданылады.

## Үйлесімділік және тексеру

Project format v20, form schema v6 және tablet layout v16 өзгерген жоқ. Қолжетімді ортада
**103 focused tests** сәтті өтті, `python -m compileall -q src tests` сәтті аяқталды. Контейнерде
PySide6, pyqtgraph және lasio болмағандықтан толық Qt/UI тексеруі орындалмады; ол жобаның
орнатылған Windows ортасында міндетті болып қалады.
