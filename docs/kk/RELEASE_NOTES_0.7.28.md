# 0.7.28 шығарылымы — бірыңғай инженерлік тор

- барлық графикалық жолдарда негізгі және аралық бөліністер біріздендірілді;
- инспекторда 1–20 негізгі және 1–20 аралық бөлініс бапталады;
- көлденең сызықтар ортақ көрінетін тереңдік/уақыт аралығында сәйкестенеді;
- тік сызықтар әр жолдың жеке X диапазоны бойынша есептеледі;
- экрандық тордан тәуелсіз «Торды басып шығару» жалаушасы қосылды;
- баптаулар планшетте, пішінде, Masterlog, PDF және физикалық баспада сақталады;
- ескі пішіндер LAS деректерін өзгертпей 5 × 5 профиліне көшіріледі;
- пішін схемасы 4-нұсқаға, планшет пішімі 14-нұсқаға жаңартылды.
