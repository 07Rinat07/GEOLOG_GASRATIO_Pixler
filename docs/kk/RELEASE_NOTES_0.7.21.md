# 0.7.21 шығарылым ескертпелері

- сілтемелер мен түсініктемелерді жылжыту/өлшемін өзгерту кезіндегі мөлдір қабат жыпылықтауы жойылды;
- тінтуір қозғалысында толық планшет емес, нысанның өзгерген аймағы ғана қайта салынады;
- QWidget native mask кэштеледі және әр mouse event кезінде қайта орнатылмайды;
- геометрия тінтуір жіберілгенде бір рет және нақты өзгеріс болғанда ғана сақталады;
- жай таңдау шертімі Undo әрекетін жасамайды және жобаны dirty күйіне ауыстырмайды;
- canvas нысандары мен image assets синхрондалғанда жолақтар, қисықтар және тақырыптар қайта құрылмайды;
- аннотация операциялары жоба ағашын жаңартпайды, себебі олар жеке қабат менеджерінде сақталады.

Жоба форматы өзгерген жоқ, бұрын жасалған аннотациялар үйлесімді.
