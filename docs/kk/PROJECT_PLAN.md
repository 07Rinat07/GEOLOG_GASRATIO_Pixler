# Жоба жоспары

Күні: 21 шілде 2026

1. **Тереңдік — GeoData** — эталон бойынша жұмыс экранын, литология, шлам, ЛБА,
   кальциметрия және мәтінді біріккен енгізу мен қайта өңдеуді, бейімделуді және сақтауды
   аяқтау. Газдың абсолюттік және салыстырмалы компоненттік құрамы іске асырылды.
2. **Masterlog** — тақырып пен дене редакторларын біріктіріп, preview, PDF және принтерге бір
   үлгіні қолдану.
3. **Gas Ratio & Pixler** — жеке шкалалар, басқарылатын интерпретация ережелері, маркерлер және
   нәтиже түсіндірмесі.
4. **Шығарылымды тексеру** — нақты LAS, Windows экрандары, физикалық баспа, миграциялар және
   RU/KK/EN құжаттамасын синхрондау.

Орындалған тармақтар [күй құжатында](PROJECT_STATUS.md) және
[тереңдік нұсқаулығында](GEODATA_DEPTH_WORKSPACE.md) берілген.

5. **Жазулар және стратиграфия** — толық жол редакторын, жобалық анықтамалықты, аралық енгізуді
   және сақтауды әр жұмыс пішінінің міндетті бөлігі ретінде қолдау.

Нақты LAS файлында Windows smoke-test орындау: параметрді таңдау, екі бағытта сызу, Undo/Redo, тәуелді қисықтарды қайта есептеу, жобаны сақтау және жаңа LAS экспорттау. Тұрақтандырылғаннан кейін көпжолақты планшет ішінде тікелей сызуды қосу.
