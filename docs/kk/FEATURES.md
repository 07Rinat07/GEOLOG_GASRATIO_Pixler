# Мүмкіндіктер мен нұсқаулықтар

Бұл файл — GEOLOG GASRATIO@Pixler пайдаланушы функцияларының өзекті картасы. Әр бөлімде
функцияның мақсаты, команданы қайдан табуға болатыны және толық нұсқаулыққа сілтеме берілген.
Ескі жинақтардың тарихы release notes файлдарында сақталады және пайдаланушы нұсқаулығын
алмастырмайды.

## Жалпы жұмыс ережелері

- Деректерге, пішіндерге, аралықтарға және аннотацияларға енгізілген өзгерістер алдымен ашық
  жоба сеансында болады.
- Оларды дискіге жазу үшін **Ctrl+S** немесе **Файл → Сақтау** командасын пайдаланыңыз.
- Өзгертілген жобаны жапқанда бағдарлама өзгерістерді сақтауды ұсынуы тиіс. Сақтамай жабу соңғы
  сақтаудан кейінгі өзгерістердің бәрін жояды.
- **Undo/Redo** қолдау көрсетілетін әрекеттерді тек ағымдағы сеанста қайтарады немесе болдырмайды.
- LAS, CSV, Excel, PDF, DOCX немесе HTML экспорты бөлек файл жасайды және жобаның өзін сақтауды
  алмастырмайды.
- Маңызды өзгерістерден кейін жобаны сақтап, жауып, қайта ашып тексеру ұсынылады.

## Енгізілген белгіні тексеру

1. Планшетті ашып, **F4** басыңыз да **«Белгі енгізу»** командасын таңдаңыз.
2. Фон нұсқасын, жолақты, параметрді немесе тереңдікке байланыстыруды, тереңдік пен өлшемді таңдаңыз.
3. Енгізгеннен кейін нысанды сол жақ батырмамен дәл орналастырып, маркерлермен өлшемін өзгертіңіз.
4. **Ctrl+S** басыңыз. Белгіге арналған жеке сақтау батырмасы жоқ: ол жобамен бірге сақталады.
5. Жобаны жауып, қайта ашыңыз. Орнын, өлшемін, тереңдігін, жолағын, параметрін және фон нұсқасын тексеріңіз.
6. Қажет болса preview/PDF тексеріңіз. PDF экспорты жобаны сақтауды алмастырмайды.

## Автоматты түрде аударылмайтын деректер

LAS мнемоникалары, өлшем бірліктері, объект идентификаторлары, пайдаланушы атаулары мен
сипаттамалары, формулалар және импортталған файлдардың мазмұны автоматты түрде аударылмайды.
Бағдарлама интерфейсі және кірістірілген пайдаланушы құжаттамасы аударылады.
