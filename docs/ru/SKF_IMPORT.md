# Импорт форм Delphi SKF

Версия 0.7.14 добавляет безопасный импорт старых файлов `.skf`, содержащих бинарный поток
компонентов Delphi.

## Что создаётся

Один SKF преобразуется в:

- редактируемый `FormDocument` для планшета;
- связанную печатную шапку `MasterlogTemplate`;
- изображения шапки, если найдены BMP, PNG или JPEG;
- отчёт импорта с именем, размером, SHA-256, корневым классом, количеством компонентов и
  предупреждениями.

Форма сохраняется в пользовательской библиотеке `forms/depth` или `forms/time`. Связанная шапка
появляется среди пользовательских печатных форм Конструктора.

## Импорт через Библиотеку форм

1. Откройте **Формы → Библиотека форм**.
2. Разверните **Импорт и экспорт**.
3. Нажмите **Импорт SKF**.
4. Выберите файл `.skf`.
5. Проверьте отчёт импорта и предупреждения.
6. Откройте созданную пользовательскую форму.

## Импорт через Конструктор

1. Откройте **Конструктор → Формы планшета**.
2. Нажмите **Импортировать форму SKF**.
3. Проверьте отчёт.
4. Откройте форму в Библиотеке форм, а связанную шапку — в разделе печатных форм.

## Сопоставление

Когда данные присутствуют в SKF, импортёр переносит:

- геометрию колонок и элементов шапки;
- заголовки, секции, выравнивание и направление текста;
- мнемоники, единицы, цвета, толщину и стиль кривых;
- линейную/логарифмическую шкалу и min/max;
- сетку;
- поля, текст, линии и изображения шапки;
- тип вертикальной оси: глубина или время.

Неизвестные пользовательские Delphi-компоненты не исполняются. Они сопоставляются с ближайшим
внутренним типом либо пропускаются с предупреждением.

## Безопасность

Импортёр не загружает Delphi-классы и не выполняет обработчики событий. Поток сначала разбирается
в нейтральное дерево. Ограничены размер файла, размер бинарных свойств, глубина вложенности и
количество компонентов. Исходный SKF открывается только для чтения.

## Сохранение и проверка результата

Импорт добавляет форму и ресурсы в текущий проект/пользовательскую библиотеку. После проверки
нажмите **Ctrl+S** и при необходимости сохраните форму отдельной командой библиотеки. Закрытие
без сохранения может отменить изменения текущего проекта. Повторно откройте форму и проверьте
колонки, шапку, шкалы, изображения, направление текста и привязки кривых. Перед печатью выполните
preflight и preview.

## Диагностика без GUI

```powershell
python tools\inspect_skf.py "C:\path\form.skf" --dump-json skf-tree.json
python tools\inspect_skf.py "C:\path\form.skf" --convert-dir converted
```

## Ограничения

Синтетический Delphi-поток покрыт тестами. Для точной проверки нестандартных классов нужны реальные
SKF-файлы исходной программы и визуальное сравнение формы, шапки и печатного результата.
