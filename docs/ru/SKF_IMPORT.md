# Импорт форм Delphi SKF

Версия 0.7.14 добавляет отдельный безопасный импортёр старых файлов `.skf`, содержащих бинарный поток компонентов Delphi.

## Что создаётся

Один SKF преобразуется одновременно в:

- редактируемый `FormDocument` для планшета;
- связанную печатную шапку `MasterlogTemplate` — это внутренняя модель HeaderTemplate проекта;
- изображения шапки, если в потоке найдены BMP, PNG или JPEG;
- отчёт импорта с именем, размером, SHA-256, классом корневого компонента, количеством компонентов и предупреждениями.

Импортированная форма сохраняется в пользовательской папке `forms/depth` или `forms/time`. Связанная шапка появляется среди пользовательских печатных форм Конструктора.

## Как импортировать

### Через Библиотеку форм

1. Откройте **Формы → Библиотека форм**.
2. Разверните **Импорт и экспорт**.
3. Нажмите **Импорт SKF**.
4. Выберите файл `.skf`.
5. Проверьте отчёт и откройте созданную пользовательскую форму.

### Через Конструктор

1. Откройте **Конструктор → Формы планшета**.
2. Нажмите **Импортировать форму SKF**.
3. После импорта форма доступна в Библиотеке форм, а шапка — в разделе печатных форм.

## Безопасность

Импортёр не загружает Delphi-классы и не выполняет обработчики событий. Поток сначала разбирается в нейтральное дерево. Ограничены размер файла, размер бинарных свойств, глубина вложенности и количество компонентов.

## Сопоставление

Импортёр переносит, когда они присутствуют в SKF:

- геометрию колонок и элементов шапки;
- заголовки, секции и направление текста;
- мнемоники, единицы, цвета, толщину и стиль кривых;
- линейную/логарифмическую шкалу и диапазон min/max;
- сетку;
- поля, текст, линии и изображения шапки;
- тип вертикальной оси: глубина или время.

Неизвестные пользовательские Delphi-компоненты не исполняются. Они преобразуются эвристически в ближайший внутренний тип либо пропускаются с предупреждением.

## Диагностика

Для анализа файла без запуска GUI:

```powershell
python tools\inspect_skf.py "C:\path\form.skf" --dump-json skf-tree.json
```

Для конвертации из установленного окружения приложения:

```powershell
python tools\inspect_skf.py "C:\path\form.skf" --convert-dir converted
```

## Ограничение проверки

Синтетический Delphi-поток проверен. Для подтверждения точного соответствия нестандартных классов старой программы необходимы реальные файлы `Глубинка.skf` и `Мастерлог ... .skf`; их следует повторно загрузить в текущий чат для контрольного импорта и визуального сравнения.
