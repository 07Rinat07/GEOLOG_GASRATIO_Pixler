# Примечания к выпуску 0.7.18

## Аннотации поверх всего планшета

- Выноски, комментарии, изображения и метки значений переведены из дорожек в единый верхний слой всей рабочей области планшета.
- Текстовый блок свободно перемещается через границы колонок; точка привязки к глубине, времени или кривой остаётся на исходных данных.
- Выбранный объект имеет восемь маркеров изменения размера: четыре угловых и четыре боковых.
- Поддерживаются одинарный щелчок для выбора, перетаскивание, двойной щелчок/F2/Enter для редактирования, Delete для удаления и контекстное меню правой кнопки.
- Кнопки «Редактировать выбранное» и «Удалить выбранное» доступны на компактной панели F4.
- Аннотации больше не заполняют дерево проекта/колонку настроек; для полного списка используется отдельный менеджер «Все…».
- Тот же верхний слой используется при захвате дорожек, PDF и печати.

## Импорт DB → LAS без зависания интерфейса

- Чтение и создание набора остаются в рабочих потоках, а построение таблиц предпросмотра разбито на короткие порции через таймер Qt.
- Удалено дорогое автоматическое вычисление ширины каждой колонки во время заполнения таблицы.
- Окно адаптируется к доступному размеру экрана, имеет системную кнопку закрытия, постоянно видимую кнопку «Отмена/Закрыть» и безопасную отмену.
- Прогресс отображается на единой шкале из шести этапов: заголовок, схема, записи, анализ, предпросмотр, создание документа.
- Показываются текущий файл, обработанные элементы, прошедшее время и подсказка для длительной операции.
- Закрытие во время чтения запрашивает отмену и ждёт безопасной границы блока, не повреждая исходный DB.
