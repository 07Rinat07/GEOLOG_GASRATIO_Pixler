# Универсальный конструктор форм, шапок и печати

Дата актуализации: 21 июля 2026  
Версия реализации: 0.7.0

## Назначение

Раздел **«Конструктор»** является единым входом для экранных форм планшета, печатных
Masterlog-форм, WYSIWYG-шапок, колонок, параметров страницы, литотипов, условных
обозначений, предварительного просмотра и проверки перед печатью. Реализация использует
существующую модель Masterlog и существующий печатный renderer, поэтому preview, PDF и
физическая печать не расходятся из-за параллельных подсистем.

- [x] используется `QGraphicsScene/QGraphicsView`, координаты и размеры задаются в миллиметрах;
- [x] поддерживаются A0–A4, Letter, Legal, пользовательский и рулонный форматы;
- [x] поддерживаются книжная и альбомная ориентации;
- [x] редактор показывает реальную ширину листа и красную область выхода элементов за границу;
- [x] доступны текст, динамическое поле, изображение, линия, литологическая и ЛБА-легенда;
- [x] изображение можно импортировать как BMP/PNG/JPEG/TIFF/WebP/SVG;
- [x] доступны режимы fit/fill/stretch, поворот и прозрачность;

### Срез 26 — глубинные обозначения и динамическая легенда

- [x] обозначение привязывается к точке глубины, интервалу, параметру кривой или времени;
- [x] выбираются колонка, параметр, изображение, размер и подпись;
- [x] добавлены ручные смещения X/Y в миллиметрах без потери исходной глубинной привязки;
- [x] renderer применяет смещения при preview, PDF и печати;
- [x] литологическая легенда поддерживает режимы «использованные», «все», «выбранные вручную» и «использованные + выбранные»;
- [x] новые проектные породы автоматически появляются в режиме «все» и могут быть выбраны вручную;
- [x] легенда раскладывается по колонкам и ограничивает содержимое своей областью;
- [ ] отдельная автоматическая легенда реально использованных условных обозначений;
- [ ] перетаскивание глубинного значка непосредственно на печатном preview с обратной записью смещения.
