# GEOLOG GASRATIO@Pixler 0.7.60

Панель статистики интервала перенесена из внешнего плавающего `QDockWidget` во внутренний
overlay планшета. Она не сжимает форму, не выходит за экран и сохраняет ручное положение при
изменении размера окна. Закрытие панели и смена формы очищают выделение и статистику.

Добавлены geometry-, contract- и Qt regression-тесты. Корневой README содержит только
описание проекта, запуск и ссылки; техническая история хранится в `docs`.

Project format v20, form schema v6 и tablet layout v16 не изменялись.
Проверка в контейнере: 19 focused tests passed; доступная headless-регрессия — 1094 passed, 4 skipped, 3 deselected. Qt-сценарии требуют установленного PySide6.
