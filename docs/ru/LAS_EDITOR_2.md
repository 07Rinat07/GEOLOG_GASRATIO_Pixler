# LAS Editor 2 — безопасное сращивание, вставка внешних данных и пакетная правка

Дата актуализации: 21 июля 2026

## Назначение

LAS Editor 2 объединяет операции, которые раньше выполнялись раздельно: нормализацию
глубины, сращивание рейсов, вставку кривых из внешнего LAS, графическую правку исходных
каналов, пакетную работу в таблице и экспорт выбранных данных.

Исходные LAS-файлы на диске не перезаписываются автоматически. Любое преобразование
создаёт новую копию в проекте либо изменяет только текущий рабочий dataset. Запись на диск
выполняется пользователем через экспорт LAS или сохранение проекта.

## Копия с возрастающей глубиной

Команда **«Создать копию с глубиной по возрастанию»** предназначена для LAS, где глубина
идёт от большего значения к меньшему. В новой копии одновременно разворачиваются:

- активный глубинный индекс;
- все кривые;
- дополнительные индексы;
- STRT, STOP и STEP;
- provenance преобразования.

Исходный dataset и исходный LAS остаются без изменений.

## Сращивание LAS по глубине

Команда **«Срастить dataset по глубине…»** объединяет два предварительно импортированных
LAS в новую производную копию. Поддерживаются три правила перекрытия:

1. **Сохранить старый LAS; новым заполнить пробелы** — основной безопасный режим.
2. **На перекрытии использовать новый LAS**.
3. **Сохранить обе версии кривых отдельно**.

Сохраняются кривые, дополнительные индексы, заголовки и параметры обоих родителей.
Несовместимые одноимённые кривые не удаляются: новая версия получает уникальную мнемонику.
В `MERGE_MANIFEST` записываются родители, политика, конфликты заголовков и различия значений.
Сращивание не интерполирует отсутствующие отсчёты и не изменяет исходные datasets.

Перед сращиванием убывающий LAS следует преобразовать в возрастающую копию. Единицы и тип
глубинного индекса обоих наборов должны совпадать.

## Вставка данных из внешнего LAS

Команда **«Вставить данные из внешнего LAS…»** читает файл с диска и добавляет выбранные
кривые непосредственно в текущий dataset. Это подходит для данных:

- геофизических исследований;
- инклинометрии и кривления скважины;
- лабораторных и инженерных измерений;
- сторонних расчётных каналов.

Окно показывает исходную мнемонику, физическую единицу, описание, количество конечных и
пустых отсчётов. Для каждой кривой задаются новая уникальная мнемоника и понятное отображаемое
название.

Правила сопоставления:

- совпадающая глубинная сетка копируется без интерполяции;
- частично совпадающая сетка интерполируется только внутри общего диапазона;
- `NULL/NaN` и крупные разрывы не соединяются;
- за пределами перекрытия остаются пропуски;
- единицы глубины `m`, `ft`, `cm`, `mm` приводятся к единице текущего LAS;
- убывающая глубина внешнего файла разворачивается только во временной копии в памяти;
- внешний файл на диске не изменяется.

В dataset сохраняется `EXTERNAL_LAS_IMPORT_*` manifest с SHA-256 источника, путём, кодировкой,
сопоставлением глубины и перечнем вставленных кривых. Вставку можно отменить, пока вставленные
кривые не были дополнительно отредактированы.

Физическая единица импортированной кривой сохраняется. Для формы и PDF понятное название и
отображаемая единица меняются в редакторе дорожки; простая смена подписи не пересчитывает
числовые значения.

## Редактирование кривой карандашом

Для правки исходной кривой:

1. Откройте на графике только одну исходную кривую.
2. Включите **«Карандаш»** или нажмите `E`.
3. Зажмите левую кнопку и проведите новую линию по нужному интервалу.
4. Отпустите кнопку.

Точки рисунка привязываются к глубинным отсчётам. Изменение выполняется одной командой
Undo/Redo. После правки синхронно пересчитываются существующие зависимые Gas Ratio,
`TG_CALC`, профили Pixler и пользовательские формулы, для которых сохранён provenance.
Расчётные кривые напрямую карандашом не редактируются — меняются их исходные параметры.

## Excel-подобная LAS-таблица

Таблица поддерживает:

- прямоугольное и несмежное выделение ячеек;
- `Ctrl+C` с табуляцией и переводами строк;
- `Ctrl+V` из Excel и других таблиц;
- вставку одного значения во множество выбранных ячеек;
- пустую ячейку как `NaN`;
- `Delete` для очистки;
- диапазонное заполнение, шум, сдвиг, умножение и сглаживание;
- атомарные Undo/Redo;
- синхронный пересчёт зависимых кривых.

Глубина и расчётные колонки защищены от прямой правки.

## Экспорт таблицы

Можно экспортировать выбранные ячейки либо все параметры в:

- Excel `.xlsx`;
- текстовый TSV `.txt`;
- CSV `.csv`.

В экспорт включаются глубина, выбранные кривые, единицы и значения. Экспорт не меняет
текущий LAS.

## Сохранение результата

- **Дискета / Ctrl+S** сохраняет проект со всеми изменениями и manifest-записями.
- **Экспорт LAS** создаёт отдельный LAS-файл по выбранному плану.
- Исходные файлы не перезаписываются без явного выбора целевого пути.
