# Tablet Engine 2.0 — навигация

Планшет использует одну камеру для всех дорожек и для осей глубины или времени.

Управление:

- колесо — прокрутка;
- `Ctrl+колесо` — масштаб относительно курсора;
- средняя кнопка или `Space + ЛКМ` — перемещение рукой;
- `Home`/`End` — начало/конец;
- `PageUp`/`PageDown` — почти на один экран;
- `↑`/`↓` — небольшой шаг;
- полоса прокрутки и переход к значению работают синхронно со всеми колонками.

Камера ограничивает окно фактическим диапазоном LAS и не позволяет дорожкам расходиться по оси.

## Горизонтальный viewport и закреплённая глубина

- дорожки сохраняют заданную ширину и больше не сжимаются по ширине окна;
- глубинная дорожка закреплена слева и не перемещается при горизонтальной прокрутке;
- остальные дорожки прокручиваются обычной горизонтальной полосой;
- изменение ширины дорожки немедленно обновляет полный горизонтальный диапазон.

## Мини-карта

Справа от планшета отображается мини-карта полного диапазона. Серое окно показывает текущий viewport. Щелчок выполняет быстрый переход, перетаскивание перемещает видимый диапазон через общую `TabletCamera`.

## Peak-preserving LOD

Для больших LAS отображается ограниченное число точек, рассчитанное по размеру viewport. В каждом сегменте сохраняются минимум и максимум, поэтому короткие пики и провалы не исчезают при уменьшении масштаба. При приближении исходные точки автоматически возвращаются.
