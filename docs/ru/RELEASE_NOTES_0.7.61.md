# GEOLOG GASRATIO@Pixler 0.7.61

## Вставка значков на график

В режиме редактирования формы **F4** добавлена отдельная команда **«Вставить значок»**. Окно
показывает встроенный справочник обозначений с миниатюрами, русскими названиями, категориями и
поиском. Пользователь выбирает:

- прозрачный обрезанный вариант без лишнего фона либо исходный вариант с фоном;
- дорожку графика;
- параметр для привязки к кривой либо привязку только к глубине;
- точную глубину и начальный размер.

После вставки значок выделяется автоматически. Его можно перетащить левой кнопкой мыши в точное
место и изменить размер восемью угловыми и боковыми маркерами. Те же действия доступны через
контекстное меню графика.

Значок копируется во внутреннее хранилище проекта и сохраняет идентификатор элемента справочника
и выбранный режим фона. Внешний путь к исходному BMP/PNG не требуется. Объект участвует в
Undo/Redo и одинаково используется экраном, PDF и печатью.

## Совместимость и проверка

Project format v20, form schema v6 и tablet layout v16 не изменялись. В доступном окружении
пройдено **103 focused tests**, `python -m compileall -q src tests` завершён успешно. Полный
Qt/UI-прогон не выполнен, потому что в контейнере отсутствуют PySide6, pyqtgraph и lasio; он
остаётся обязательным в установленном Windows-окружении проекта.
