# Руководство пользователя

GEOLOG GASRATIO@Pixler — редактор буровых, газокаротажных и LAS-данных.

## Язык

При первом запуске выберите русский, қазақша или English. Позже язык можно изменить через
меню «Язык / Language / Тіл». Для согласованного обновления всех окон перезапустите приложение.

## Импорт

Используйте «Файл → Импортировать данные...» (`Ctrl+I`) и выберите LAS, CSV/TXT или Excel.
Оригинальные файлы не изменяются.

Полная русская инженерная документация находится в родительском каталоге `docs` и постепенно
переносится в синхронные пользовательские руководства RU/KK/EN.
