# Примечания к выпуску 0.7.62 — аудит документации на трёх языках

Версия 0.7.62 улучшает документацию и автоматическую проверку качества. Project format v20,
form schema v6 и tablet layout v16 не изменялись.

## Выполнено

- добавлена актуальная карта пользовательских функций со ссылками на подробные инструкции;
- объяснено единое сохранение данных, форм, интервалов, аннотаций и значков через проект;
- инструкция по значкам дополнена сохранением, закрытием, повторным открытием, Undo/Redo,
  удалением, блокировкой, флагом печати, PDF и внутренним хранением изображения;
- синхронизированы пользовательское руководство, статус, план, release notes и навигация;
- добавлена автоматическая проверка одинакового набора документов RU/KK/EN, внутренних ссылок,
  ключей локализации, версии пакета и обязательных сценариев сохранения/повторного открытия;
- добавлены regression-тесты документационного контракта;
- корневой README оставлен кратким, без подробной истории исправлений.

## Ограничение проверки

Аудит документации прошёл для 82 файлов на язык и 1881 i18n-ключа. Доступный headless-набор:
**1103 passed, 4 skipped, 3 deselected**. Полный сбор блокируют 82 модуля с PySide6/pyqtgraph/
lasio; Windows Qt/UI, LAS, PDF и физическая печать остаются обязательными.
