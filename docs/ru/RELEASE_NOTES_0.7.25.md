# Release notes 0.7.25 — пакетная настройка DB и безопасный рендер планшета

Статус: **тестовая сборка** до обязательной визуальной проверки на Windows.

## Пакетный DB → LAS

- неоднозначный индекс больше не считается обычной ошибкой;
- строка получает статус **«Требуется настройка»**;
- кнопка **«Настроить выбранный DB…»** открывает стандартное окно GeoScape/Paradox;
- пользователь выбирает глубину, время, каналы и единицы без открытия отдельного документа;
- выбранный план хранится в текущем сеансе и применяется к глубинному и временному экспорту;
- после настройки предлагается повторить только выбранный файл;
- реальные ошибки, пропуски существующих файлов и запрос настройки визуально разделены.

## Планшет и аннотации

- удалён полноразмерный прозрачный `QWidget`, который на Windows мог перекрывать PyQtGraph чёрным прямоугольником;
- `TabletAnnotationOverlay` теперь является скрытым менеджером геометрии и сигналов и ничего не рисует сам;
- каждая видимая аннотация рендерится в отдельный небольшой `QLabel`-спрайт с прозрачным `QPixmap`;
- спрайт ограничен фактическими границами объекта и телом графиков ниже шапки;
- пустые участки планшета не имеют поверх себя дочернего окна аннотаций;
- маршрутизация мыши, редактирование дорожек, Undo/Redo и печатный рендер сохранены.

## Проверка

В контейнере выполнены компиляция, модульные и source-contract тесты, проверка локализаций и реальных Paradox-образцов. Реальный `PySide6 + pyqtgraph` Windows-рендер в контейнере недоступен, поэтому сборка не объявляется стабильной.
