# Примечания к выпуску 0.7.16

Версия добавляет полноценный путь GeoScape/Paradox DB → единая `Dataset`-модель → существующий редактор → проект/LAS/графики/планшеты. Основные изменения: безопасный бинарный reader, поиск DB/PX/TV/FAM, кандидаты глубины/времени, профили, словарь каналов, контроль качества, временной и глубинный LAS, TIME → DEPTH, пакетный конвертер, прогресс и отмена.

Проверенные образцы: `BLData.db` — 3488 записей/70 полей, `D250.db` — 1739/101. Исходные файлы не изменены. Известное ограничение релизной проверки: в текущем контейнере отсутствует `lasio`, поэтому фактический повторный импорт созданного LAS должен быть выполнен в штатной Windows-среде.
