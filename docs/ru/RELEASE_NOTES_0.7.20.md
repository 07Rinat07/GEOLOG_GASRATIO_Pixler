# Примечания к выпуску 0.7.20

## Единое отображение даты и времени

- Введён один форматтер `ДД.ММ.ГГГГ ЧЧ:ММ:СС`, одинаковый на Windows и Linux.
- Курсор планшета, таблица LAS, отдельное окно кривых, предварительный просмотр Paradox, редактор аннотаций и печатные инспекции больше не показывают календарное время как необъяснимое число секунд.
- Поддерживаются `numpy.datetime64`, Unix seconds/milliseconds, Delphi/OLE Automation date и относительное время.
- При наличии календарного индекса или даты начала относительное время показывается как полная дата и время; без начала регистрации используется `ЧЧ:ММ:СС`.
- Из исходного кода пользовательского отображения удалены платформозависимые `datetime.fromtimestamp` и `utcfromtimestamp`.

## Прямой режим аннотаций F4

1. Включите F4.
2. Выберите «Выноска», «Комментарий» или «Изображение».
3. Щёлкните по точной дорожке и глубине/времени.
4. Объект создаётся сразу в выбранной точке.
5. Перетаскивайте блок за фон/рамку и меняйте размер восемью маркерами.
6. Двойной щелчок, F2, Enter, кнопка панели или контекстное меню открывают полный редактор.

Аннотационный слой теперь отсечён по телу графиков: рамка, текст, линия и маркеры не могут проходить поверх шапки дорожки или подписей параметров. При создании у верхней/нижней границы начальное положение автоматически выбирается внутри видимой области.

## Совместимость

Формат проекта и схема аннотаций не изменены. Проекты 0.7.15–0.7.19 открываются без миграции. Исходные LAS/DB не изменяются.
