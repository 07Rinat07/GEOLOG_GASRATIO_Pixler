# Статус проекта

Срез: 23 июля 2026 года. Версия 0.7.30, тестовая сборка.

## Решение о выпуске

Последний полный baseline 0.7.28 зелёный: Ruff проходит, mypy сообщает 0 ошибок в
262 исходных файлах, полный pytest — 1217 пройдено и 10 пропущено. Для 0.7.30 выполнены
`compileall`, сборка wheel и 73 headless/regression/source-integrity тестов для session binding,
workspace-команд, project controllers и печатной границы. Полный gate нужно повторить в
установленном окружении, потому что текущий контейнер не содержит PySide6, pyqtgraph, lasio,
Ruff и mypy, а package index не предоставляет эти пакеты. Сборка остаётся тестовой до полного
gate и Windows/HiDPI/PDF/физической печати.

## Рабочая основа

- безопасный импорт и редактирование LAS/CSV/TXT/Excel/Paradox;
- проекты с несколькими datasets и индексами;
- многотрековый планшет, интервалы, литология и формы;
- Masterlog, PDF, Print Center и настраиваемые крупные/мелкие сетки;
- выделение интервала `Shift + ЛКМ`, статистика и экспорт XLSX/CSV;
- аннотации, project assets и миграции старых проектов;
- синхронные пользовательские документы RU/KK/EN.

Печатные операции выполняются через `PrintJobExecutor`. Новый `SessionBindingController`
перепривязывает 26 контроллеров и сбрасывает их Undo/Redo или временный выбор. Команды дерева
проекта проходят через `WorkspaceCommandController`, поэтому `MainWindow` больше не меняет
`current_well_id/current_dataset_id` в обработчике дерева. Следующий срез — запрет прямой
мутации сериализуемой модели проекта из UI.
