# Статус проекта

Срез: 23 июля 2026 года. Версия 0.7.29, тестовая сборка.

## Решение о выпуске

Последний полный baseline 0.7.28 зелёный: Ruff проходит, mypy сообщает 0 ошибок в
262 исходных файлах, полный pytest — 1217 пройдено и 10 пропущено. Для 0.7.29 выполнены
компиляция исходников и расширенный регрессионный набор для import jobs, адаптеров и
project-session boundary: 105 тестов пройдено, 3 платформенных теста пропущены; wheel пакета 0.7.29 собран без ошибок. Полный gate нужно повторить в установленном окружении, потому что текущий
контейнер не содержит PySide6, pyqtgraph, lasio, Ruff и mypy, а package index недоступен.
Сборка остаётся тестовой до полного gate и Windows/HiDPI/PDF/физической печати.

## Рабочая основа

- безопасный импорт и редактирование LAS/CSV/TXT/Excel/Paradox;
- проекты с несколькими datasets и индексами;
- многотрековый планшет, интервалы, литология и формы;
- Masterlog, PDF, Print Center и настраиваемые крупные/мелкие сетки;
- выделение интервала `Shift + ЛКМ`, статистика видимых параметров и экспорт XLSX/CSV;
- аннотации, project assets и миграции старых проектов;
- синхронные пользовательские документы RU/KK/EN.

`ImportJobController` и `DatasetImportJobExecutor` находятся в `services/import_jobs.py`.
Единая граница выполняет CSV/Excel планы, применяет strict/compatible/manual политику LAS,
сохраняет lossless source/import report и регистрирует Paradox dataset. `MainWindow` только
собирает выбор пользователя и показывает результат; отклонённый или ошибочный файл не создаёт
частично зарегистрированный dataset. Следующий срез — print jobs и session binding.
