# Статус проекта

Срез: 24 июля 2026 года. Версия пакета: **0.7.61**. Project format: **v20**,
form schema: **v6**, tablet layout: **v16**.

## Завершено в 0.7.61

- добавлена отдельная команда вставки значков из встроенного справочника на графики планшета;
- окно выбора показывает миниатюры, локализованные названия, категории и поиск;
- поддерживается выбор прозрачного обрезанного изображения или исходного варианта с фоном;
- перед вставкой задаются дорожка, параметр/привязка к глубине, точная глубина и начальный размер;
- вставленный объект можно перемещать левой кнопкой мыши и масштабировать восемью маркерами;
- значок сохраняется во внутреннем хранилище проекта вместе с catalog ID и режимом фона;
- существующие Undo/Redo, сохранение проекта, PDF и печать используются без параллельного формата;
- документация и инструкции синхронизированы на русском, казахском и английском языках;
- корневой README не содержит технической истории релиза.
