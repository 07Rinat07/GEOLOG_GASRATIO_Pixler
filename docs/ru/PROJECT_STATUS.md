# Статус проекта

Срез: 24 июля 2026 года. Версия пакета: **0.7.46**, аварийная тестовая сборка. Project format: **v20**.

## Завершённый hotfix 0.7.46

- устранена регрессия PySide6 в grid overlay: вместо целого `0` используется `Qt.MouseButton.NoButton`;
- ошибка отрисовки сетки больше не блокирует уже импортированный LAS;
- при сбое планшета программа открывает безопасный табличный режим восстановления;
- Import Review предупреждения остаются неблокирующими;
- импорт собирает единый диагностический отчёт по этапам чтения, разбора, policy, review, регистрации и отображения;
- отчёт содержит код, severity, источник, рекомендуемое исправление, context, exception type и traceback;
- блокирующие отчёты автоматически сохраняются в application data, а диалог поддерживает копирование и ручное сохранение;
- повторяющиеся мнемоники читаются по физическим столбцам; один повреждённый канал не отклоняет весь LAS;
- добавлен regression-сценарий 9 847 строк / 73 канала, соответствующий проблемному классу файлов.

## Сохранённая рабочая основа

Project format v20, form schema v6 и tablet layout v15 не менялись. Несколько независимых
DEPTH/TIME datasets, рабочие формы, обозначения, комментарии, настройки кривых и ежедневный
append LAS сохраняют прежний контракт.
