Срез: 24 июля 2026 года. Версия пакета: **0.7.52**. Project format: **v20**, form schema:
**v6**, tablet layout: **v16**.

## Завершено в 0.7.52

- исправлена ошибка `Internal C++ object (CurveHeaderEditor) already deleted` при импорте,
  reset планшета и переключении форм;
- каждый QObject проверяется через Shiboken до удаления event filter или `deleteLater()`;
- очистка треков продолжается после ошибки одного устаревшего Qt wrapper и безопасна при
  повторном вызове;
- редактируемая шапка кривой уменьшена с 82 до 52 пикселей, обычная подпись — до 38 пикселей;
- минимум, единица, максимум, тип шкалы и инженерная линейка сохранены в компактном виде;
- общий пояс шапок ограничен 360 пикселями, поэтому колонки с малым числом параметров содержат
  значительно меньше пустого места;
- исправлены предупреждения `QFont` с размером `-1` и некорректная начальная нормализация цветов;
- LAS-дубликаты, неравномерный шаг и разрывы остаются неблокирующими предупреждениями с
  конкретными неразрушающими рекомендациями.
