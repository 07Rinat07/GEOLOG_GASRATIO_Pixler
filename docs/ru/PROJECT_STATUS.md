# Статус проекта

Срез: 23 июля 2026 года. Версия 0.7.28, тестовая сборка.

## Решение о выпуске

Автоматическая часть релизного gate зелёная: Ruff проходит, mypy сообщает 0 ошибок в
258 исходных файлах, полный pytest — 1193 пройдено и 10 пропущено, код завершения 0.
Сборка остаётся тестовой до ручной проверки Windows/HiDPI/PDF/физического принтера.

## Рабочая основа

- импорт и безопасное редактирование LAS/CSV/TXT/Excel/Paradox;
- проекты с несколькими datasets и индексами;
- многотрековый планшет, интервалы, литология и формы;
- Masterlog, PDF, Print Center и настраиваемые крупные/мелкие сетки;
- выделение интервала `Shift + ЛКМ`, статистика всех видимых параметров и экспорт XLSX/CSV;
- аннотации, project assets и миграции старых проектов;
- пользовательские документы RU/KK/EN.

Восстановлена маршрутизация аннотаций, устранены Windows-блокировка SVG и несовместимость
PNG-буфера PySide6. Панели планшета теперь адаптируются к узкому экрану.

Annotation event router, навигация и order/reuse треков вынесены из `TabletView`.
Перестановка и Undo/Redo больше не пересоздают графики. Следующая точка: подписанный
Windows smoke-test, затем создание/удаление треков. Подробнее:
[аудит](PRODUCT_AUDIT_2026.md) и [план](PROJECT_PLAN.md).
