# Статус проекта

Срез: 24 июля 2026 года. Версия пакета: **0.7.59**. Project format: **v20**,
form schema: **v6**, tablet layout: **v16**.

## Завершено в 0.7.59

- по диагностическому traceback устранено падение `TabletTrackWidget` без `_localizer`;
- каждый трек получает активный локализатор `TabletView` до формирования шапки;
- прямое создание трека также безопасно использует русский fallback-localizer;
- плотные формы с более чем шестью параметрами строят локализованную подсказку прокрутки без `AttributeError`;
- транзакционный откат формы остаётся рабочим и не заменяет последнюю успешную компоновку;
- добавлены regression-контракты и Qt-тест сценария dense form switch.

## Проверка

В контейнере выполнены `compileall` и доступные pure/source-contract тесты. Полный Windows/PySide6
smoke-test обязателен: последовательно применить формы с 7–12 параметрами в одной колонке на RU/KK/EN.
