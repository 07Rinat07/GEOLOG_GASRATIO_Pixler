# Примечания к выпуску 0.7.21

- устранено мерцание прозрачного слоя при перетаскивании и изменении размера выносок/комментариев;
- во время движения перерисовывается только изменённый прямоугольник объекта, а не весь планшет;
- нативная маска QWidget кэшируется и не переустанавливается на каждом событии мыши;
- изменение геометрии фиксируется только после отпускания мыши и только при реальном смещении/изменении размера;
- простой щелчок для выбора больше не создаёт Undo-команду и не помечает проект изменённым;
- синхронизация canvas-объектов и image assets больше не перестраивает дорожки, кривые и шапки;
- дерево проекта не обновляется при операциях с аннотациями, поскольку объекты хранятся в отдельном менеджере слоя.

Формат проектов не изменён. Ранее созданные аннотации совместимы.
