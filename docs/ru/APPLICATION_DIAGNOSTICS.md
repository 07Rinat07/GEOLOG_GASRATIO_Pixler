# Диагностика приложения

Начиная с версии 0.7.51 GEOLOG GASRATIO@Pixler постоянно ведёт вращаемый журнал.

Команда **«Справка → Открыть папку журналов»** открывает каталог. Текущий файл называется
`geolog.log`, предыдущие версии сохраняются как нумерованные ротации. Файл `geolog-crash.log`
предназначен для faulthandler и сведений об аварийном завершении.

При проблеме используйте **«Справка → Собрать пакет диагностики…»**. ZIP содержит:

- текущий и несколько предыдущих журналов;
- версию приложения, Python и сведения об операционной системе;
- безопасный runtime-контекст: идентификатор dataset, количество треков, состояние карандаша и
  транзакции формы.

Значения LAS, проектные assets, сохранённые формы и файлы проекта не добавляются. В обычных
сообщениях журнала могут присутствовать пути, названия проекта и dataset, поэтому перед передачей
пакета сторонним лицам его следует проверить.

Для воспроизводимого отчёта повторите проблемное действие, сразу создайте пакет и приложите ZIP
вместе с кратким описанием последовательности нажатий.
