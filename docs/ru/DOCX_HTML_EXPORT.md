# Экспорт выбранного интервала в DOCX и HTML

Версия 0.7.40 добавляет два новых формата в меню **Файл**:

- **Экспортировать выделение в DOCX** — редактируемый документ Microsoft Word;
- **Экспортировать выделение в HTML** — один автономный файл для браузера и печати.

## Как экспортировать

1. Откройте dataset.
2. Выделите интервал глубины на графике или в таблице.
3. Выберите необходимые кривые.
4. В меню **Файл** выберите DOCX или HTML.
5. Укажите имя файла и подтвердите перезапись при необходимости.

Оба формата используют тот же интервал, что CSV и XLSX. Программа не считывает viewport
повторно и не заменяет выбранный индекс другим.

## Что находится в документе

- название dataset и отчёта;
- точные границы и число отсчётов;
- ReportDefinition SHA-256;
- таблица покрытия каналов;
- таблица значений выбранного интервала.

Обозначения:

- `0` — реальный измеренный ноль;
- `—` — пропущенный отсчёт;
- `#N/A` — запрошенный канал отсутствует в dataset.

HTML не требует интернета и не загружает внешние скрипты или стили. DOCX не содержит
макросов и внешних объектов.

Рядом с файлом создаётся `*.report-passport.json`. Паспорт содержит размер, MIME и SHA-256
готового DOCX/HTML. Если файл изменить после экспорта, проверка паспорта обнаружит это.
