# GEOLOG GASRATIO@Pixler 0.7.52 — безопасная очистка Qt и компактные шапки

Исправление выполнено по диагностическому пакету пользователя от 24 июля 2026 года.

## Исправление lifecycle

- перед обращением к QObject проверяется `shiboken6.isValid()`;
- удаление event filter и `deleteLater()` безопасно пропускают уже уничтоженный C++-объект;
- очистка остальных треков продолжается, даже если один устаревший wrapper вызвал исключение;
- импорт и переключение форм больше не должны блокироваться ошибкой `CurveHeaderEditor already deleted`;
- повторный reset планшета является идемпотентным.

## Компактная шапка

- высота редактируемой шапки уменьшена с 82 до 52 пикселей;
- в первой строке находятся название, линейный/логарифмический режим, авто-диапазон и настройки;
- минимум, единица и максимум размещены в одной компактной строке;
- инженерная линейка использует сохранённые основные и промежуточные деления;
- обычная подпись кривой уменьшена до 38 пикселей;
- общий синхронизированный пояс ограничен 360 пикселями вместо 480.

## Предупреждения LAS

Дубликаты индекса, неравномерный шаг и разрывы остаются предупреждениями. Для них показываются конкретные варианты: политика дублей, отдельная ресэмплированная копия с шагом 0,2 м либо сохранение пропусков. Исходный LAS не перезаписывается.
