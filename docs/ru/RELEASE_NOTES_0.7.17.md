# Примечания к выпуску 0.7.17

## Назначение выпуска

Hotfix исправляет реальные отказы профессионального слоя аннотаций, обнаруженные при ручной проверке версии 0.7.16. Формат проекта и модель сохранённых аннотаций не менялись.

## Исправлено

- кнопки «Выноска», «Комментарий» и «Изображение» панели F4 снова открывают редактор;
- создание из панели и контекстного меню использует понятное окно «Сохранить/Отмена»;
- текст, стиль, смещение, ширина и высота редактируются до сохранения;
- существующий блок перемещается мышью, изменяет размер нижним правым маркером и открывается двойным щелчком;
- правая кнопка над объектом открывает его контекстное меню, а не меню дорожки;
- новая выноска имеет заметную линию между привязкой и текстовым блоком;
- стандарт GeoScape 0,2 м показывается отдельно от фактического шага DB. Для переданного BLData фактический LAS `STEP` остаётся 0,4 м до явного ресэмплинга.

## Совместимость

Проекты 0.7.15/0.7.16 открываются без миграции. Исходные LAS/DB не изменяются. Обновление заменяет только код приложения и документацию.
