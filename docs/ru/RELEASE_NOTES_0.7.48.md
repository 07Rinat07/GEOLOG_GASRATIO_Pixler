# GEOLOG GASRATIO@Pixler 0.7.48 — инженерная шкала и единицы в шапке кривой

Тестовая сборка от 24 июля 2026 года. Project format остаётся v20, form schema — v6,
tablet layout повышен с v15 до v16.

## Шапка параметра

- минимум и максимум остаются редактируемыми непосредственно в шапке;
- добавлена видимая цветная линия шкалы с основными и промежуточными делениями;
- положения делений совпадают с нормализованной вертикальной сеткой конкретной колонки;
- промежуточные подписи вычисляются отдельно для линейной и логарифмической шкалы;
- значения сначала можно подготовить в обоих полях, затем применить кнопкой `✓` или Enter;
- кнопка `A` возвращает автоматический диапазон.

## Единица и тип шкалы

- единица измерения видна и редактируется прямо в шапке;
- линейная/логарифмическая шкала переключается там же;
- единица является настройкой отображения и не пересчитывает исходные числовые значения;
- полное окно параметров содержит ту же редактируемую единицу;
- пользовательская единица, диапазон, тип шкалы, цвета и деления сохраняются в рабочей форме;
- старые tablet layout v15 безопасно мигрируют в v16 с `unit_override = null`, то есть продолжают
  использовать единицу из metadata исходного канала.
