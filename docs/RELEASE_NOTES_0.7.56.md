# GEOLOG GASRATIO@Pixler 0.7.56 — контроль ширины формы и адаптивная статистика

- Добавлен единый расчёт экранной ширины формы относительно полезной ширины книжного и альбомного A4 при 96 DPI.
- Библиотека форм показывает число колонок, ширину в px/мм, ожидаемый процент масштаба и конкретную рекомендацию.
- Редактор структуры обновляет A4-индикатор сразу при добавлении, удалении или изменении ширины колонки.
- В предпросмотре редактора отображаются границы книжного и альбомного A4, если форма их превышает.
- Строка состояния планшета постоянно показывает: форма помещается, лучше альбомная ориентация, нужен Fit или форму следует разделить.
- Панель статистики интервала больше не остаётся плавающим окном за правым краем: на широком экране она закрепляется справа, на узком — снизу.
- Для нижнего режима таблица статистики получает более широкие числовые столбцы и компактную высоту; для бокового — узкую адаптивную компоновку.
- Диагностический отчёт теперь содержит ширину текущей формы и проценты A4 portrait/landscape.
- Package **0.7.56**; project format **v20**; form schema **v6**; tablet layout **v16**.
