# План проекта GEOLOG GASRATIO@Pixler

## Правила структуры

- `README.md` остаётся в корне и содержит знакомство, запуск и автора.
- вся подробная документация хранится строго в `docs/`;
- реализация ведётся небольшими рабочими инкрементами;
- каждый инкремент сопровождается тестами и обновлением документации;
- обязательный quality gate и матрица проверок описаны в `docs/TESTING.md`;
- исходный LAS не перезаписывается скрытно.

## Реализованные этапы

### 0.1–0.2 — базовое ядро

- модели проекта, скважины, набора данных и кривой;
- импорт LAS;
- проектная сессия;
- базовые Gas Ratio;
- безопасное атомарное сохранение;
- реестр методик Pixler;
- тесты вычислительного ядра.

### 0.4 — Tablet Engine Foundation

- отдельный модуль `tablet`;
- модель `TrackDefinition` и `TabletLayout`;
- базовые типы треков;
- многотрековый экран планшета;
- синхронизация треков по общей оси глубины;
- горизонтальная прокрутка;
- построение базовой компоновки по открытому LAS;
- трек глубины;
- газовый трек;
- треки отдельных кривых;
- переключение dataset из Project Explorer;
- инспектор выбранного трека;
- модель глубинного viewport для будущих zoom/pan и виртуализации.
