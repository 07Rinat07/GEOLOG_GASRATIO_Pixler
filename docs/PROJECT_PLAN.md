# План проекта GEOLOG Gas Ratio & Pixler

Дата актуализации: 20 июля 2026

Языковые версии: [Русский](ru/PROJECT_PLAN.md) · [Қазақша](kk/PROJECT_PLAN.md) · [English](en/PROJECT_PLAN.md)

## Цель версии 0.6.x

Стабилизировать ежедневный рабочий процесс ГТИ: импортировать неоднородные LAS, распознавать
параметры, отображать их в редактируемой глубинной или временной форме, вводить геологические
интервалы, выполнять газовые расчёты и печатать Masterlog без потери исходных данных.

## Этап 1 — глубинная рабочая форма GeoData

- [x] курируемая библиотека из трёх заводских форм;
- [x] секции формы и объединённые заголовки по эталону GeoData;
- [x] точный порядок колонок без принудительного закрепления глубины;
- [x] общая высота шапки и синхронная глубинная область;
- [x] горизонтальная прокрутка на ноутбуках;
- [x] сворачиваемый правый инспектор;
- [x] сохранение секций и компоновки в layout v10.

## Этап 2 — ручной ввод геологии

- [x] создание литологии через `Shift + левая кнопка`;
- [x] повторная правка и удаление литологического интервала;
- [x] создание общей пробы через шламограмму/ЛБА/кальциметрию;
- [x] до четырёх пород с суммой 100%;
- [x] ЛБА: тип, цвет и интенсивность 1–5 в трёх подколонках;
- [x] кальцит, доломит и автоматически рассчитываемый остаток;
- [x] rich-text описание, вставка символов и изображений;
- [x] атомарное повторное редактирование по `sample_id`;
- [x] сохранение по дискете и `Ctrl+S`.

## Этап 3 — Masterlog

- [x] отдельная заводская геолого-геохимическая форма;
- [x] интервальные литология, шламограмма, ЛБА и кальциметрия;
- [x] компонентный газ, технологические параметры и описание пород;
- [x] многостраничный PDF и универсальный центр печати;
- [ ] единый визуальный редактор шапки и тела формы;
- [ ] единый пользовательский шаблон для экрана, preview, PDF и принтера;
- [ ] полный каталог редактируемых литологических, газовых и технологических символов;
- [ ] контроль визуального соответствия на A4/A3 и рулоне.

## Этап 4 — Gas Ratio & Pixler

- [x] базовые отношения, расчётный суммарный газ и относительный компонентный состав;
- [x] семантическое распознавание C1–C5 и единиц;
- [x] отдельные шкалы и корректные разрывы на отсутствующих данных;
- [ ] специализированная рабочая форма Gas Ratio/Pixler;
- [ ] управляемая классификация газ/влажный газ/конденсат/нефть/неопределённо;
- [ ] объяснение каждого маркера и оценка качества входов;
- [ ] ручное подтверждение, корректировка и журнал изменений;
- [ ] печатная интерпретационная легенда.

## Этап 5 — качество и выпуск

- [ ] проверка на реальных LAS разных поставщиков;
- [ ] матрица экранов 1366×768, 1920×1080, 2560×1440 и HiDPI;
- [ ] Windows-проверка физической печати, включая принтеры с портом `FILE:`;
- [ ] производительные бюджеты на 100 тыс., 1 млн и 5 млн отсчётов;
- [ ] установщик и миграционная проверка старых проектов;
- [ ] финальная синхронизация документации RU/KK/EN.

## Ограничения плана

До отдельного решения не включаются WITSML, DLIS, 3D-моделирование, облачная синхронизация и
многоскважинная корреляция. Такие задачи не должны задерживать завершение рабочих форм ГТИ.
