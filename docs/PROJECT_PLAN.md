## P0 — 0.7.58: статистика интервала без расширения окна

- [x] вносить изменение непосредственно в полное дерево проекта, без обязательного patch workflow;
- [x] держать статистику как плавающее overlay-окно поверх планшета;
- [x] не включать ширину статистики в minimum size главного окна;
- [x] ограничивать окно рабочей областью активного монитора;
- [x] повторно позиционировать overlay при перемещении и изменении размера главного окна;
- [x] сохранять доступными заголовок, закрытие, таблицу и кнопки экспорта;
- [x] закрытие overlay очищает выделение интервала и dataset selection;
- [x] смена формы очищает старую заливку и статистику до перестроения widget tree;
- [x] смена dataset/проекта также очищает старый анализ;
- [x] покрыть геометрию независимыми headless-тестами и source-contract-тестами;
- [ ] Windows/PySide6: проверить 1366×768, 1600×900, 1920×1080, два монитора и DPI 100/125/150%.

Критерий выхода: форма сохраняет всю доступную ширину; статистика может перекрывать правую
часть графиков, но не выходит за монитор; после закрытия или смены формы не остаётся старого
выделения.
