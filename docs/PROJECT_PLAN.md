# План реализации GEOLOG GASRATIO@Pixler

## Правила структуры

- В корне проекта находится только `README.md` для знакомства, запуска и автора.
- Вся расширенная документация и инструкции находятся строго в `docs/`.
- Локальные виртуальные окружения, пакеты, кэш, пользовательские проекты и отчёты исключаются `.gitignore`.

## Текущий статус — версия 0.3

Реализовано:

- жизненный цикл проекта: новый, открыть, сохранить, сохранить как;
- безопасное восстановление проекта из JSON;
- проверка типов, длины глубины и кривых при открытии;
- загрузка нескольких LAS за одну операцию;
- переключение между наборами данных в Project Explorer;
- выбор и просмотр отдельной кривой;
- инспектор проекта, скважины, набора и кривой;
- предупреждение о несохранённых изменениях;
- автор и контакты в README и окне «О программе»;
- расширенный `.gitignore`;
- тест сериализации проекта.

## Следующий инкремент 0.4 — многотрековый планшет

1. Общая вертикальная шкала глубины.
2. Несколько независимых треков рядом.
3. Выбор кривых для каждого трека.
4. Линейная и логарифмическая шкала.
5. Изменение ширины треков.
6. Синхронное масштабирование и прокрутка.
7. Сохранение компоновки в проекте.
8. Отдельный газовый трек C1–C5 и Total Gas.
9. Подготовка архитектуры инструмента «Карандаш».

## Дальнейшие инкременты

- 0.5: интерактивное редактирование кривых и Undo/Redo;
- 0.6: шламограмма, литология и стратиграфия;
