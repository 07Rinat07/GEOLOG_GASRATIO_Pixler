# Инкремент 0.2 — первый рабочий вертикальный срез

Реализовано:

1. Открытие LAS из GUI.
2. Создание скважины и набора данных в проектной сессии.
3. Просмотр нескольких кривых.
4. Определение базовых газовых мнемоник через aliases.
5. Расчёт C1/C2, C1/C3, C2/C3, C1/(C2+C3), TG_CALC.
6. Добавление расчётных кривых обратно в dataset.
7. Атомарное сохранение проекта.
8. Безопасный реестр профилей Pixler без непроверенных встроенных формул.
9. Тесты расчётов и графа зависимостей.

Следующий инкремент:

- выбор кривых в интерфейсе;
- несколько треков с общей глубиной;
- редактор мнемоник;
- подтверждённый профиль рабочей формулы Pixler;
- Undo/Redo для изменения кривых;
- первый режим карандаша;
- собственный формат проекта SQLite + массивы.
