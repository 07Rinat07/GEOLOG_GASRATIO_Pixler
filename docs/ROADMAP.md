# Roadmap

- **0.1–0.2 Core/Data**: проект, LAS, базовые Gas Ratio, атомарное сохранение.
- **0.4 Tablet Foundation**: многотрековый планшет и общая шкала глубины — реализовано.
- **0.5 Track Manager**: выбор кривых, геометрия, шкалы и сохранение layout — реализовано.
- **0.6 Interactive Editing**: карандаш, Undo/Redo и экспорт нового LAS — реализовано.
- **0.7 Gas/Pixler/DEXP**: подтверждённые формулы, provenance и dependency graph.
- **0.8 Geology**: литология, шламограмма, ЛБА, стратиграфия, керн и интервальные объекты.
- **0.9 Catalog Studio**: породы, литотипы, редактор узоров и профили организаций.
- **0.10 Correlation**: несколько скважин, GR, реперы и горизонты.
- **0.11 Print Studio**: шапки, формы, PDF, принтер, интервалы и вся скважина.
- **0.12 Documentation Center**: формулы, источники, проверочные примеры и Plugin API.
- **1.0 Stable**: миграции, восстановление, производительность, установщик и шаблоны.
