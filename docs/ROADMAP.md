# Roadmap

- **0.1 Data Core**: проект, импорт LAS, диагностика, каталоги мнемоник, безопасное сохранение.
- **0.2 Workspace**: главное окно, Project Explorer, Inspector, Issues, треки одной скважины.
- **0.3 Gas/DEXP**: C1–C5, Total Gas, ratios, Pixler, DEXP, dependency graph.
- **0.4 Editing**: карандаш, пересчёт, Undo/Redo, экспорт нового LAS.
- **0.5 Geology**: литология, шламограмма, ЛБА, стратиграфия, керн, интервальные объекты.
- **0.6 Catalog Studio**: породы, литотипы, редактор узоров и профили организаций.
- **0.7 Correlation**: несколько скважин, GR, реперы, горизонты.
- **0.8 Print Studio**: шапки, формы, PDF, принтер, интервалы и вся скважина.
- **0.9 Documentation**: формулы, источники, проверочные примеры и Plugin API.
- **1.0 Stable**: миграции, восстановление, производительность, установщик и эталонные шаблоны.
