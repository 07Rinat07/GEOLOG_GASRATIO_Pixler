# Инкремент 0.4 — Tablet Engine Foundation

## Цель

Перейти от одиночного графика к базовому геологическому планшету, где несколько
треков используют единую глубинную шкалу и могут прокручиваться горизонтально.

## Реализовано

- модуль `geoworkbench.tablet`;
- `TrackKind`, `TrackDefinition`, `TabletLayout`;
- `DepthViewport` с ограниченным pan/zoom;
- `TabletView` на основе PyQtGraph;
- связанная ось Y между всеми треками;
- трек глубины;
- газовый трек;
- треки отдельных кривых;
- автоматическая базовая компоновка;
- добавление базовых треков через меню;
- выбор трека и показ его свойств;
- переключение dataset двойным щелчком в дереве;
- открытие нескольких LAS за одну операцию;
- обновление версии до 0.4.0.

## Технические ограничения текущего инкремента

- компоновка ещё не сериализуется;
- отсутствует диалог точного выбора кривых;
- ширина и порядок треков пока меняются только программно;
- литология, шламограмма и стратиграфия ещё не отрисовываются;
- виртуализация больших массивов будет добавлена после менеджера треков.

## Следующий шаг

Реализовать полноценный Track Manager: выбор кривых, ширина, порядок, скрытие,
удаление, шкалы X и сохранение макета планшета.
