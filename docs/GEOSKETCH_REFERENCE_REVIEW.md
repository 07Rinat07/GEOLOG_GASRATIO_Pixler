# Обзор GeoSketch II как функционального референса

Изучено руководство пользователя GeoSketch II версии 1.0.0 (ООО «ГеоСенсор»,
2021, 321 страница). Оно используется только для анализа рабочих процессов;
защищённые текст, изображения, справочники и оформление не копируются.

## Полезные направления

1. Объектная модель документа и дерево компонентов.
2. Комментарии, закладки и реперы с привязкой к глубине.
3. Выравнивание объектов и управляемый порядок слоёв.
4. Визуальное создание литологических интервалов с запретом пересечений.
5. Единая история Undo/Redo для данных и объектов.
6. Шаблоны геологических описаний по тематическим разделам.
7. Раздельные режимы просмотра и редактирования макета.
8. Компоненты для литологии, шлама, керна, стратиграфии и корреляции.

## Реализовано по итогам обзора

Добавлен редактор глубинных заметок. Заметки сохраняются в существующих
`Well.canvas_objects` как объекты типа `depth_annotation`, имеют устойчивый ID,
глубину и текст. Контроллер проверяет пустой текст, конечность глубины и попадание
в диапазон текущего dataset.
