# GEOLOG GASRATIO@Pixler 0.7.60

Исправлена повторная регрессия панели статистики выделенного интервала. Нативное плавающее
окно `QDockWidget` зависело от рамки Windows, DPI и нескольких мониторов, поэтому могло
оказаться за правой границей и возвращалось вправо после ручного перемещения.

Панель заменена дочерним overlay-компонентом внутри планшета. Она может перекрывать правую
часть графиков, но не изменяет ширину формы и не может выйти за границы родительской рабочей
области. Ручное положение сохраняется при resize; закрытие и переключение формы полностью
очищают интервал, shading, dataset selection и отчёт.

Добавлены независимые geometry-тесты, source-contract проверки и Qt regression-сценарии.
Корневой README очищен от описаний конкретных hotfix, traceback и результатов тестов; такие
сведения остаются в документации каталога `docs`.

Project format v20, form schema v6 и tablet layout v16 не изменялись.
Проверка в контейнере: 19 focused tests passed; доступная headless-регрессия — 1094 passed, 4 skipped, 3 deselected. Qt-сценарии требуют установленного PySide6.
