# GEOLOG GASRATIO@Pixler 0.7.61

На графиках планшета появилась отдельная команда **«Вставить значок»**. Она открывает
локализованный справочник обозначений с поиском, названиями и категориями. Перед вставкой можно
выбрать прозрачный обрезанный вариант либо исходное изображение с фоном, дорожку, параметр,
глубину и начальный размер.

Значок сохраняется как внутренний ресурс проекта вместе с идентификатором справочника и режимом
фона. После вставки его можно точно перемещать левой кнопкой мыши и изменять размер восемью
маркерами. Используется существующая модель аннотаций, поэтому объект поддерживает Undo/Redo,
сохранение проекта, PDF и печать.

Project format v20, form schema v6 и tablet layout v16 не изменялись. Проверка в доступном
окружении: 103 focused tests passed; `compileall` выполнен успешно. Полный Qt/UI-прогон требует
установленных PySide6, pyqtgraph и lasio.
