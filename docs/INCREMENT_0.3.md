# Инкремент 0.3 — Project Lifecycle

## Выполнено

- команды «Новый проект», «Открыть проект», «Сохранить» и «Сохранить как»;
- атомарная запись проекта;
- безопасная десериализация без `pickle`;
- лимит размера открываемого JSON-проекта;
- проверка длины кривых относительно шкалы глубины;
- загрузка нескольких LAS;
- переключение текущего dataset через дерево;
- просмотр отдельной кривой и её метаданных;
- подтверждение потери несохранённых изменений;
- единый номер версии 0.3.0;
- обновлённый README и автор проекта;
- расширенный `.gitignore`.

## Ограничения

JSON-снимок пока хранит числовые массивы непосредственно в файле. Для крупных
промышленных проектов он будет заменён на SQLite для метаданных и отдельное
бинарное/колоночное хранилище массивов.
