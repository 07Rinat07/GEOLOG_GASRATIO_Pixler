# Инкремент 0.6 — Interactive Curve Editor

## Цель

Добавить безопасное интерактивное редактирование кривых без скрытого изменения исходного LAS.

## План

1. [x] Командная модель изменения отдельных отсчётов.
2. [x] Ограниченная история Undo/Redo с обнаружением конфликтов.
3. [ ] Application-controller редактирования и состояния зависимых кривых.
4. [ ] Инструмент карандаша в Qt/PyQtGraph.
5. [ ] Команды Undo/Redo в меню и горячие клавиши.
6. [ ] Экспорт результата в новый LAS-файл.
7. [ ] Интеграционные и Qt-тесты пользовательского редактирования.

## Инварианты

- исходный LAS никогда не перезаписывается;
- команда хранит копии только изменённых значений;
- индексы уникальны и находятся в границах кривой;
- внешнее изменение значений обнаруживается как конфликт;
- execute, undo и redo увеличивают версию кривой.
