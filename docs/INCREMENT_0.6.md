# Инкремент 0.6 — Interactive Curve Editor

## Цель

Добавить безопасное интерактивное редактирование кривых без скрытого изменения исходного LAS.

Статус: реализовано в версии 0.6.0.

## План

1. [x] Командная модель изменения отдельных отсчётов.
2. [x] Ограниченная история Undo/Redo с обнаружением конфликтов.
3. [x] Application-controller редактирования и состояния зависимых кривых.
4. [x] Инструмент карандаша в Qt/PyQtGraph.
5. [x] Команды Undo/Redo в меню и горячие клавиши.
6. [x] Экспорт результата в новый LAS-файл.
7. [x] Интеграционные и Qt-тесты пользовательского редактирования и экспорта.

## Инварианты

- исходный LAS никогда не перезаписывается;
- команда хранит копии только изменённых значений;
- индексы уникальны и находятся в границах кривой;
- внешнее изменение значений обнаруживается как конфликт;
- execute, undo и redo увеличивают версию кривой.
- зависимые выходы после edit, undo и redo переходят в состояние `STALE`.
- карандаш доступен только при выборе одной кривой и не меняет данные напрямую.
- экспорт атомарен и никогда не перезаписывает исходный LAS.
- импорт и экспорт LAS не требуют optional-зависимости pandas.
