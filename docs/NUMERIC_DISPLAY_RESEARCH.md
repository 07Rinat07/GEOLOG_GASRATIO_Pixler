# Форматы чисел в отраслевых программах

Исследование выполнено по официальной документации аналогов перед реализацией формата чисел
LAS-таблицы.

- Techlog описывает интерактивное редактирование данных в spreadsheet-представлении:
  <https://www.slb.com/es/products-and-services/delivering-digital-at-scale/software/techlog-wellbore-software/techlog/techlog-core-systems/techlog-base>.
- WellCAD позволяет ограничивать число отображаемых цифр в табличном виде:
  <https://www.alt.lu/download/wellcad-5-7-build-2017-release-notes/>.
- DUG Insight позволяет задавать число знаков после запятой для числовых колонок:
  <https://help.dugeo.com/m/Insight/l/1535333-using-table-to-overlay-views>.
- Seequent разделяет округлённое представление отчёта и экспорт с полной точностью:
  <https://help.seequent.com/Energy/2026.1/en-GB/Content/estimation/resource-report.htm>.

Из этого следует принятая модель:

1. Формат выбирается для одной или нескольких колонок, а не изменяет весь dataset.
2. По умолчанию используется читаемый адаптивный десятичный режим без `E±n`.
3. Для отчётных колонок доступно фиксированное число знаков, для специалистов — явный
   scientific-режим.
4. Округляется только экран. Редактирование и экспорт используют исходное число полной
   точности.
5. Настройка привязана к устойчивой мнемонике и профилю инженера, поэтому применяется к LAS
   другой фирмы после сопоставления параметров.
