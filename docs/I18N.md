# Локализация RU / KK / EN

## Обязательная политика

GEOLOG GASRATIO@Pixler поддерживает русский, казахский и английский языки. Выбор предлагается
при первом запуске и сохраняется в `QSettings`; изменить его можно через меню
«Язык / Language / Тіл». После смены приложение просит перезапуск, чтобы одновременно
обновить все открытые окна.

Каждая новая пользовательская строка должна:

1. иметь стабильный смысловой ключ;
2. одновременно добавляться в `ru.json`, `kk.json` и `en.json`;
3. использовать единый отраслевой термин из глоссария;
4. применяться также в отчётах и печатных формах;
5. сопровождаться тестом равенства ключей трёх каталогов.

Мнемоники LAS (`DEPT`, `C1`, `ROP`, `WOB`), единицы, формулы, имена файлов и исходные данные
не переводятся. Переводится их описание и окружающий интерфейс.

## Проверенный базовый глоссарий

| Русский | Қазақша | English |
|---|---|---|
| Скважина | Ұңғыма | Well |
| Глубина | Тереңдік | Depth |
| Месторождение | Кен орны | Field |
| Оператор | Оператор | Operator |
| Бурение | Бұрғылау | Drilling |
| Буровой раствор | Бұрғылау ерітіндісі | Drilling fluid / mud |
| Пласт | Қабат | Formation / reservoir layer |
| Пластовое давление | Қабаттық қысым | Formation pressure |
| Горная порода | Тау жынысы | Rock |
| Долото | Қашау | Drill bit |
| Ствол скважины | Ұңғыма оқпаны | Wellbore |
| Забой скважины | Ұңғыма кенжары | Bottomhole |
| Каротаж | Каротаж | Well logging |
| Кривая | Қисық | Curve |
| Отчёт | Есеп | Report |
| Печатная форма | Баспа нысаны | Print form |

Контекст важен: английское `formation` может означать геологическую формацию или пласт,
а `mud` допустим как краткая промысловая форма `drilling fluid`. Мнемоники и утверждённые
международные сокращения не локализуются.

а не публикуется как окончательный автоматически.

## Документация

Исходные инженерные документы пока ведутся на русском. Для пользовательских руководств
создаются синхронные версии `docs/ru`, `docs/kk`, `docs/en`. Изменение инструкции считается
завершённым только после обновления трёх версий либо явной отметки `translation pending` в
проверке документации. Формулы, код и ссылки на стандарты должны совпадать между версиями.

## Текущее покрытие

- оболочка: меню, вкладки, панели, выбор и сохранение языка;
- универсальный импорт: общая точка входа;
- CSV/TXT и Excel: заголовки, mapping, preview, проверки и кнопки;
- LAS Table Editor: подсказки, диапазонные команды, диалоги и проверки selection;
- LAS Export Plan: версия, WRAP, NULL, точность, подтверждения и статус экспорта;
- Formula Profiles: паспорт, источник, mapping и описания DEXP на RU/KK/EN;
- Interval Statistics: меню, интервал, показатели и проверки на RU/KK/EN;
- Depth Annotations: редактор, действия, проверки и дерево проекта на RU/KK/EN;
- Lithology Intervals: редактор, действия, проверки и дерево проекта на RU/KK/EN;
- Description Templates: редактор, действия, проверки и дерево проекта на RU/KK/EN;
- Lithology Legend: заголовки, пустое состояние и названия RU/EN на RU/KK/EN;
- Lithotype Catalog: таблица, форма, действия и тип источника на RU/KK/EN;
- Track Inspector: сводка, шкала, диапазон и свойства кривой на RU/KK/EN;
- Data Inspector: все вкладки, индексы, кривые, диагностика и LAS-заголовок на RU/KK/EN;
- печать: базовый словарь терминов для будущего Print Studio.

LAS import review, редакторы метаданных, геологические диалоги и специализированные сообщения пока
мигрируют по плану. Отсутствие перевода не должно скрываться: статус фиксируется в roadmap.

Каталог литотипов сейчас хранит названия RU/EN. В казахском интерфейсе до расширения схемы
используется русское справочное название рядом с неизменным `lithotype_id`; пользовательские
описания интервалов никогда не переводятся автоматически.
