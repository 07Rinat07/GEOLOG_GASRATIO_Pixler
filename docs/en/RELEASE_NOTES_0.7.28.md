# Release notes 0.7.28 — unified engineering grid

- all graphical tracks now use consistent major and minor divisions;
- the inspector supports 1–20 major divisions and 1–20 minor subdivisions;
- horizontal lines remain aligned on the shared visible depth/time interval;
- vertical lines follow each track's individual X range;
- a separate Print grid option keeps screen guidance independent from output;
- settings survive tablet/form round trips and linked Masterlog/PDF/physical printing;
- legacy forms migrate to the safe 5 × 5 profile without modifying LAS data;
- form schema is version 4 and tablet layout is version 14.
