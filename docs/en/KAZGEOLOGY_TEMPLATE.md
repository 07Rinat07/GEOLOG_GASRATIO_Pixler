# “MASTERLOG — reference header and depth form” template

The template follows the supplied Masterlog reference and is configured for A3 landscape output.

## Form contents

- two-level survey/Masterlog title band;
- independent uploadable contractor and customer logo slots;
- country, region, district/block, field and well information;
- project, customer, contractor, drilling company and target formation;
- interval, scale, rig, well type and personnel;
- actual/planned depth, drilling dates, coordinates and LAS source;
- lithology legend, LBA legend, symbols and well construction;
- stratigraphy, drilling, depth, cuttings, LBA, calcimetry, lithology, gas and manual rock-description columns.

## Create the form

1. Open **Constructor → Masterlog templates**.
2. Select **Create from preset**.
3. Choose **“MASTERLOG — reference header and depth form”**.
4. Enter a name for the user copy.

The built-in preset remains unchanged. All edits are stored in the created user copy.

## Load logos

1. Open the header editor.
2. Double-click **Load contractor logo** or **Load customer logo**.
3. Select **Import image**.
4. Choose PNG, JPEG, BMP, TIFF, WebP or SVG.
5. Select `fit`, `fill` or `stretch`, then save the element.

Empty logo slots are valid: preflight does not block output and the placeholder remains visible until an image is selected.

## Header data, scales and grid

Use **Header data…** to fill editable metadata. Project, well, LAS source, interval and sample count can be resolved from the active project automatically.

Drilling, calcimetry and gas curves have independent colours, minima, maxima, major divisions and minor divisions. The column editor stores these settings in the user template.

The **Rock and cuttings description** column is manual-only; lithology text is not inserted automatically.
