# GEOLOG GASRATIO@Pixler 0.7.57 — плавающая статистика интервала

- Панель статистики открывается поверх правой части планшета и больше не отнимает ширину у формы.
- Размер и положение панели ограничиваются рабочей областью экрана; кнопка закрытия и нижние действия остаются доступными.
- При смене формы очищаются старое выделение интервала, синяя заливка, dataset selection и статистика предыдущей формы.
