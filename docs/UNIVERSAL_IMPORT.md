# Универсальный импорт

Команда «Файл → Импортировать данные...» (`Ctrl+I`) является общей точкой входа для всех
реализованных локальных форматов. На первом шаге пользователь выбирает источник:

- LAS 1.2/2.0 — политика strict/compatible/manual и выбор одного или нескольких файлов;
- CSV/TXT — кодировка, разделитель, индекс, DATE+TIME и timezone;
- Excel XLS/XLSX/XLSM — лист, строка заголовка, индекс, DATE+TIME и timezone.

После выбора открывается специализированный безопасный диалог формата. Такое разделение
оставляет общую навигацию простой, но не смешивает несовместимые параметры LAS, CSV и Excel.
Старые команды «Импортировать LAS/CSV/Excel» сохранены для быстрых рабочих сценариев.

Все адаптеры создают общую доменную модель `Dataset`, не изменяют оригинал и добавляют данные
в проект только после успешной проверки и подтверждения диалога.
