# Универсальный импорт

Команда «Файл → Импортировать данные...» (`Ctrl+I`) является общей точкой входа для всех
реализованных локальных форматов. На первом шаге пользователь выбирает источник:

- LAS 1.2/2.0 — политика strict/compatible/manual и выбор одного или нескольких файлов;
- CSV/TXT — кодировка, разделитель, индекс, DATE+TIME и timezone;
- Excel XLS/XLSX/XLSM — лист, строка заголовка, индекс, DATE+TIME и timezone;
- GeoScape/Paradox — выбор dataset и безопасная регистрация извлечённых каналов.

После форматного диалога открывается единый Import Review. Пользователь проверяет активный
индекс, NULL и QC, может исключить канал и исправить canonical mapping, quantity class или UOM.
Блокирующие ошибки не позволяют подтвердить импорт, а предупреждения остаются видимыми.
Старые команды «Импортировать LAS/CSV/Excel» сохранены для быстрых рабочих сценариев.

Все адаптеры создают общую доменную модель `Dataset`, не изменяют оригинал и передают в проект
только валидированную копию после одной атомарной команды подтверждения. Отмена Import Review
не создаёт скважину или dataset и не меняет `dirty`. Подробнее: [Import Review](IMPORT_REVIEW.md).
