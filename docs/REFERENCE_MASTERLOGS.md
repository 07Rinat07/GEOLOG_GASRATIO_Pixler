# Эталонные мастерлоги

В требования включены решения, выявленные в двух предоставленных примерах:

- `Литология_Култук №16_29.08.2014_34-3155м.pdf`;
- `Мастерлог Каражар_KR-1_инт 20-3250м.pdf`.

Общие блоки: шапка, легенда пород, ЛБА, конструкция скважины, технологические кривые, глубина, шламограмма, стратиграфия, литология, карбонатность, газовые кривые, описание пород, керн, газ-тесты, маркеры и события.

Они используются как функциональные эталоны, но не копируются жёстко: любая такая компоновка должна собираться конструктором шаблонов.
