# Брендинг приложения

Основной логотип хранится в `src/geoworkbench/resources/geologist-logo.png`. Изображение
предоставлено владельцем проекта и распространяется вместе с приложением на условиях
Proprietary License проекта.

Ресурс загружается через `importlib.resources`, поэтому работает из исходного дерева и после
установки wheel. `pyproject.toml` явно включает `resources/*.png` в package data.

Логотип используется:

- как иконка `QApplication` и главного окна;
- в диалоге «О программе»;
- в дальнейшем может использоваться установщиком и splash screen без копирования в код.

Оригинальный PNG хранится без перерисовки. Qt создаёт масштабированные представления с
сохранением пропорций и сглаживанием; исходный файл при этом не изменяется.
